Como ampliar o raio de alcance de sua rede sem fio

Você pode configurar vários AirPort Extremes ou estações base AirPort Express como um WDS ( Wireless Distribution System) para ampliar o raio de alcance de sua rede sem fio AirPort. Quando você conecta as estações base sem fio a um WDS, é possível configurar cada estação base como uma estação base principal, uma estação base remota ou uma estação base de retransmissão.

Nota: Se você estiver ampliando o raio de alcance de uma rede sem fio 802.11n, clique no link abaixo para obter instruções simplificadas.

Como ampliar o raio de alcance de uma rede 802.11n

Uma estação base principal é conectada à Internet e compartilha sua conexão com estações base remotas e de retransmissão. Uma estação base remota simplesmente compartilha a conexão à Internet da estação base principal. Uma estação base de retransmissão compartilha a conexão à Internet da estação base principal e transfere a conexão a outras estações base remotas.

As configurações de todas as três estações base (principal, remota e de retransmissão) também podem compartilhar a conexão à Internet da estação base principal com computadores cliente de modo sem fio, ou por meio de uma conexão de Ethernet se os computadores cliente estiverem conectados à estação base por meio de Ethernet.

Quando você configura estações base em um WDS, precisa saber o ID do AirPort de cada estação base. O ID do AirPort, também conhecido como endereço MAC (Media Access Controller), está impressa na etiqueta da parte inferior da estação base AirPort Extreme, ao lado do símbolo AirPort, e ao lado do adaptador de alimentação da estação base AirPort Express. Para facilitar a configuração de um WDS, coloque todas as estações base em uma mesa e conecte-as a uma fonte de alimentação.

Para começar a configurar um WDS, abra o Utilitário AirPort, selecione seu dispositivo sem fio e, em seguida, escolha Estação Base > Ajuste Manual. Digite a senha, se necessário. Clique em AirPort na barra de ferramentas e, em seguida, clique em Sem Fio.

O utilitário AirPort está localizado na pasta Utilitários da pasta Aplicativos em um computador que utiliza o Mac OS X e em Iniciar > Todos os programas > AirPort em um computador que utiliza o Windows.

No painel Sem Fio:

  1. Escolha “Participar de uma rede WDS” no menu local Modo Sem Fio.

  2. Clique em WDS.

  3. Siga um destes procedimentos:

    • Se estiver configurando uma estação base principal WDS, clique no botão Adicionar (+) e digite o ID do AirPort ou o endereço MAC Sem Fio das estações base remotas WDS.

    • Se estiver configurando uma estação base remota, digite o ID do AirPort ou o endereço MAC Sem Fio da estação base principal WDS.

    • Se estiver configurando uma estação base de retransmissão, digite o ID do AirPort ou o endereço MAC sem fio da estação base principal WDS e, em seguida, clique em Adicionar (+) para inserir os IDs do AirPort ou os endereços MAC sem fio das estações base remotas WDS.

  4. Utilize o mesmo nome de rede e senha para todas as estações base no WDS.

  5. Clique em AirPort na barra de ferramentas, clique em Sem Fio e utilize o mesmo canal para cada estação base do WDS.

Por padrão, a opção “Permitir clientes sem fio” está selecionada. Se você desmarcar a opção “Permitir clientes sem fio nesta estação base” e, mais tarde, quiser alterar as configurações de sua estação base, será preciso conectar-se à porta LAN da estação base utilizando um cabo Ethernet. Talvez você não possa conectar-se à estação base utilizando o modo sem fio sem reconfigurar a estação.

Se suas estações base estiverem conectadas usando Ethernet, consulte “Como usar várias estações base em uma rede Ethernet” para ampliar o alcance da rede sem fio.
