设定基站或 Time Capsule 的 DHCP 选项

如果您已经将基站或 Time Capsule 设置为使用动态主机配置协议 (DHCP) 来将 IP 地址提供给无线网络上的电脑和设备，您可以选取设备使用的 IP 地址范围、确定电脑 IP 地址的有效时间、输入欢迎信息以及保留特定 IP 地址以分配给加入网络的特定电脑。

打开“AirPort 实用工具”（在 Mac 上，它位于“应用程序”文件夹的“实用工具”文件夹中；在运行 Windows 的电脑上，它位于“开始”>“所有程序”>“AirPort”中），在列表中选择设备，然后点按“手动设置”。如有必要，请输入密码。在工具栏中点按“互联网”，然后点按“DHCP”。

您可以设定以下 DHCP 选项：

  • 从“DHCP 范围”弹出式菜单中选取一个 IP 地址范围。根据您想要基站提供的地址，选取 10.0、192.168 或 172.16，然后在“DHCP 开始地址”栏和“DHCP 结束地址”栏中分别输入开始地址和结束地址。

  • 在“DHCP 租期”栏中输入一个数字，然后从弹出式菜单中选取“分钟”、“小时”或“天”。

  • 在“DHCP 信息”栏中键入欢迎信息。电脑加入网络时将显示此信息。

  • 若要将特定 IP 地址提供给无线网络上的特定电脑，请点按“DHCP 保留”列表下面的添加按钮 (+)，然后按照屏幕指示给保留的地址命名。按 MAC 地址或 DHCP 客户端 ID 来保留地址。如果选取 MAC 地址，请点按“继续”并输入 MAC 地址和特定的 IP 地址。

如果网络被设置为使用网络上的轻量级目录访问协议 (LDAP) 服务器，则可以在“LDAP 服务器”栏中输入服务器的地址，并且网络上的电脑将有权访问它。
