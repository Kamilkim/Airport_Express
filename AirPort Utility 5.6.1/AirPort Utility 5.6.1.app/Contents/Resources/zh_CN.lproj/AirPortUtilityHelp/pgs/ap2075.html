共享互联网连接

您可以将基站或 Time Capsule 设置为使用动态主机配置协议 (DHCP) 和网络地址转换 (NAT) 来与网络上的所有客户端共享单个 IP 地址。

如果基站或 Time Capsule 使用 PPPoE 连接到互联网，它已经使用 DHCP 和 NAT 共享单个 IP 地址了。

若要共享互联网连接：

  1. 打开“AirPort 实用工具”（在运行 Mac OS 的电脑上，位于“应用程序”文件夹的“实用工具”文件夹中；在运行 Windows 的电脑上，则位于“开始”>“所有程序”>“AirPort”中）。

  2. 选择您正在设置的设备，然后点按“手动设置”。如有必要，请输入基站密码。

  3. 在工具栏中点按“互联网”，然后点按“互联网连接”。

  4. 从“连接共享”弹出式菜单中选取“共享公共 IP 地址”或“分配一个 IP 地址范围”。

通过共享公共 IP 地址来启用 DHCP 和 NAT，或者分配一个 IP 地址范围让基站或 Time Capsule 动态并自动地将 IP 地址分配给客户端电脑，这会简化每台电脑的 TCP/IP 配置。
