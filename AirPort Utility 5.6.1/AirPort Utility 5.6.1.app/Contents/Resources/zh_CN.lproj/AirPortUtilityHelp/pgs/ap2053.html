使用以太网连接到互联网

如果您使用 DSL 或线缆调制解调器连接到互联网，或者使用局域网 (LAN) 连接到互联网，您可以将基站或 Time Capsule 连接到调制解调器或网络，并且与无线网络上的电脑和客户端共享互联网连接。

您需要设定基站或 Time Capsule 如何从互联网服务商 (ISP) 处或网络接收互联网地址，以及设备如何与网络上的电脑共享互联网连接。

请打开“AirPort 实用工具”，选择您想要设置的设备，然后点按“手动设置”。如有必要，请输入密码。在工具栏中点按“互联网”，然后点按“互联网连接”。从“连接使用”弹出式菜单中选取“以太网”，然后点按“TCP/IP”。

在运行 Mac OS X 的电脑上，“AirPort 实用工具”位于“应用程序”文件夹的“实用工具”文件夹中；在运行 Windows 的电脑上，“AirPort 实用工具”位于“开始”>“所有程序”>“AirPort”中。

若要将设备设置为使用以太网进行连接：

  1. 通过选取以下一项来设置无线设备如何使用“配置 IPv4”弹出式菜单来从 ISP 处接收 IP 地址：

    • 如果 ISP 自动发送 IP 地址给您，则请选取“使用 DHCP”。

    • 如果 ISP 向您提供了一个包括 IP 地址的帐户，则请选取“手动”。输入您从 ISP 收到的信息，例如 IP 地址、子网掩码、路由器地址及其他信息（如果适用的话）。

  2. 点按“连接共享”并设置设备与网络上的电脑共享互联网连接的方式；或者如果它连接在网络上，将它设置为网桥。

    • 从“连接共享”弹出式菜单中选取“共享公共 IP 地址”，以便使用 DHCP 和 NAT 与网络上的电脑共享您的 IP 地址。

    • 如果 ISP 向您提供了一个 IP 地址范围，则请从“连接共享”弹出式菜单中选取“分配一个 IP 地址范围”。

    • 如果设备已连接到具有能够分配 IP 地址的 DHCP 服务器的网络，请选取“关闭（网桥模式）”。

  3. 点按“更新”以将设备复位，并使用这些新设置。
