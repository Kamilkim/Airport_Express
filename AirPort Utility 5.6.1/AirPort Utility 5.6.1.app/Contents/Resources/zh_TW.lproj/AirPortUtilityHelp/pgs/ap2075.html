共享您的 Internet 連線

您可以將基地台或 Time Capsule 設定為，使用“動態主機配置協定”（DHCP）及“網路位址轉譯”（NAT）來與網路上的所有用戶端共享單一 IP 位址。

若您的基地台或 Time Capsule 是使用 PPPoE 連接 Internet，則它已經使用 DHCP 和 NAT 來共享單一 IP 位址。

若要共享您的 Internet 連線：

  1. 請打開“AirPort 工具程式”，它位於使用 Mac OS X 電腦的“應用程式∕工具程式”檔案夾裡，而在 Windows 電腦上，它位於 [開始] > [所有程式] > AirPort。

  2. 選取您要設定的設備，然後按一下“手動設定”。輸入基地台的密碼（如有需要）。

  3. 在工具列中，按一下 Internet，然後按一下“Internet 連線”。

  4. 請從“連線共享”彈出式選單裡選擇“共享一個公用 IP 位址”或“分配一組 IP 位址範圍”。

透過共享一個公用 IP 位址來啟用 DHCP 和 NAT，或分配一組 IP 位址，皆會使基地台或 Time Capsule 動態且自動分配 IP 位址給用戶端電腦，從而簡化每台電腦的 TCP/IP 設定。
