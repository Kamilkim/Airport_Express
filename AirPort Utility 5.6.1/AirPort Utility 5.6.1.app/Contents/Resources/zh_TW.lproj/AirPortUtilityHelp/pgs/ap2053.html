使用乙太網路來連接 Internet

若您使用 DSL 或纜線數據機來連接 Internet，或者您使用“區域網路”（LAN）來連接 Internet，您可以將基地台或 Time Capsule 連接至數據機或網路，然後與您的無線網路上的電腦和用戶端共享 Internet 連線。

您需要設定基地台或 Time Capsule 如何從 Internet 網路服務供應商（ISP）或網路接收 Internet 位址，以及設備如何與網路上的電腦共享 Internet 連線。

打開“AirPort 工具程式”，選取您要設定的設備，然後按一下“手動設定”。輸入密碼（如有需要）。在工具列中按一下 Internet，然後按一下“Internet 連線”。從“連接方式”彈出式選單中選擇“乙太網路”，然後按一下 TCP/IP。

“AirPort 工具程式”位於 Mac OS X 電腦的“應用程式∕工具程式”檔案夾裡，而在 Windows 電腦上，它位於 [開始] > [所有程式] > AirPort 裡。

若要設定設備為使用乙太網路連接：

  1. 透過使用“設定 IPv4”彈出式選單並選擇以下之一，來設定無線設備如何從 ISP 接收 IP 位址：

    • 若您的 ISP 為您提供自動 IP 位址，請選擇“使用 DHCP”。

    • 若您的 ISP 是依照帳號來為您提供 IP 位址，請選擇“手動”。請輸入您從 ISP 收到的資訊，如 IP 位址、子網路遮罩、路由器位址及其他相應資訊。

  2. 按一下“連線共享”並設定您的設備與網路上電腦共享 Internet 連線的方式，或將其設定為在連接到網路時作為橋接器。

    • 從“連線共享”彈出式選單中，選擇“共享一個公用 IP 位址”，以使用 DHCP 及 NAT 來與您網路上電腦共享 IP 位址。

    • 若您 ISP 提供了一組 IP 位址，請從“連線共享”彈出式選單中，選擇“分配一組 IP 位址範圍”。

    • 若設備連接到由 DHCP 伺服器分配 IP 位址的網路，請選擇“關（橋接模式）”。

  3. 按一下“更新”，來使用這些新設定重置設備。

若您是使用“透過乙太網路使用點對點協定”（PPPoE）帳號連接 Internet，請按一下下方的連結。
