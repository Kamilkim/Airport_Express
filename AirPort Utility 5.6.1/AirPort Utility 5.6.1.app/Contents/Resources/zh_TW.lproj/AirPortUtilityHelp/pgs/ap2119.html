設定基地台或 Time Capsule 的 DHCP 選項

若您設定基地台或 Time Capsule 使用“動態主機配置協定”（DHCP）來為無線網路上的電腦和設備提供 IP 位址，您可以選擇設備使用的 IP 位址範圍、決定電腦將會擁有的有效 IP 位址的時間、輸入歡迎訊息，以及保留特定的 IP 位址來指定給加入您網路的特定電腦。

打開“AirPort 工具程式”（位於 Mac 的“應用程式∕工具程式”檔案夾裡，而在使用 Windows 的電腦內，則位於 [開始] > [所有程式] > AirPort 裡），選取列表中的設備，然後按一下“手動設定”。輸入密碼（如有需要）。按一下工具列中的 Internet，然後按一下 DHCP 。

您可以設定以下 DHCP 選項：

  • 從“DHCP 範圍”彈出式選單中選擇 IP 位址範圍。請選擇 10.0、192.168 或 172.16，然後依照您想要基地台提供的位址，在“DHCP 開始位址”和“DHCP 結束位址”分別輸入開始和結束位址。

  • 請在“DHCP 租約”欄位中輸入數字，然後從彈出式選單中選擇分鐘、小時或天。

  • 在“DHCP 訊息”欄位輸入一個歡迎訊息。當有電腦加入您的網路，此訊息便會顯示。

  • 若要為無線網路上的特定電腦提供特定的 IP 位址，請在“DHCP IP 保留”列表下按一下“加入”（+），然後依照螢幕說明來為保留命名。依照 MAC 位址或 DHCP 用戶端識別碼來保留位址。若您選擇 MAC 位址，請按一下“繼續”，然後輸入 MAC 位址和特定 IP 位址。

若您的網路設定為使用“輕量型目錄存取通訊協定”（Lightweight Directory Access Protocol，LDAP）伺服器，您可以在“LDAP 伺服器”欄位輸入伺服器的位址，您網路上的電腦將會連接到它。
