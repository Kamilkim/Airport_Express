Protezione del network wireless mediante password

Per proteggere le comunicazioni sul network wireless, abilita la crittografia dati sulla base AirPort o su Time Capsule. Agli utenti che accedono al tuo network wireless viene richiesto di inserire una password oppure un nome utente e una password.

Esistono diversi tipi di password che possono essere scelti per proteggere il network, in base al dispositivo che stai configurando e alle funzionalità dei computer client che accedono al network.

WEP: se la base lo consente, scegli questa opzione e inserisci una password di tipo WEP (Wired Equivalent Privacy) per proteggere il network. Alcune basi AirPort Extreme e alcune basi AirPort Express supportano la crittografia a 40 bit e 128 bit. Scegli “40-bit WEP” dal menu a comparsa Sicurezza wireless per ottenere la massima compatibilità oppure "128-bit WEP" per fornire maggiore sicurezza WEP.

Se scegli la crittografia a 128 bit, solo i computer forniti di schede network wireless con questa crittografia potranno collegarsi al network. Se scegli la crittografia a 40 bit, i computer forniti di schede network wireless con crittografia a 128 bit e a 40 bit potranno collegarsi, ma sfrutteranno solo la crittografia a 40 bit.

WPA/WPA2 Personal: scegli questa opzione per proteggere il network con Wi-Fi Protected Access. Scegli Password, quindi inserisci una password che contenga tra 8 e 63 caratteri ASCII oppure scegli "Chiave pre-condivisa" e inserisci una password di esattamente 64 caratteri esadecimali. I computer client wireless che utilizzano WPA o WPA2 possono accedere al network.

WPA2 Personal: se al network accedono solo computer client che utilizzano WPA2, scegli questa opzione per proteggere il network con Wi-Fi Protected Access.

WPA/WPA2 Enterprise: scegli questa opzione se stai configurando un network che include un server di autenticazione, come un server RADIUS (Remote Authentication Dial-In User Service). Inserisci l'indirizzo IP e il numero di porta per il server principale e secondario quindi inserisci una "chiave condivisa" che sarà la password del server. Utilizza questa opzione se accederanno al network computer client wireless che utilizzano WPA e WPA2.

WPA2 Enterprise: scegli questa opzione se stai configurando un network che comprende un server di autenticazione al quale accederanno solo computer client che utilizzano WPA2.

WEP (Transitional Security Network): utilizza questa opzione se entrambi i computer compatibili WEP e WPA/WPA2 accederanno al network. Utilizza una password esattamente di 13 caratteri.

Per proteggere il network AirPort wireless con una password:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo wireless che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic su AirPort nella barra degli strumenti, quindi su Wireless.

  4. Scegli uno schema di password dal menu a comparsa "Sicurezza wireless".

  5. Inserisci la password che sarà richiesta a tutti gli utenti per poter accedere a questo network wireless.

Se scegli di non indicare alcuna password, il network wireless sarà accessibile a chiunque si trovi nel raggio di azione del network.
