Connettersi a Internet tramite Ethernet

Se utilizzi un modem DSL, un modem via cavo oppure un network di area locale (LAN, Local Area Network) per connetterti a Internet, puoi collegare la base o Time Capsule al modem o al network e condividere la connessione Internet con i computer e i client presenti sul network wireless.

Devi impostare la modalità utilizzata dalla base o da Time Capsule per ricevere l'indirizzo Internet dal tuo ISP (Internet Provider) o network e quello utilizzato dal dispositivo per condividere la connessione Internet con i computer del network.

Apri Utility AirPort, seleziona il dispositivo da configurare, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password. Fai clic su Internet nella barra degli strumenti, quindi su Connessione Internet. Scegli Ethernet dal menu a comparsa “Connessione via”, quindi fai clic su TCP/IP.

Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

Per impostare il dispositivo per la connessione tramite Ethernet:

  1. Configura la modalità in cui il dispositivo riceve l'indirizzo IP dal tuo Internet Provider utilizzando il menu a comparsa Configura IPv4 e scegliendo una delle seguenti opzioni:

    • Scegli "Utilizza DHCP" se il tuo Internet Provider invia un indirizzo IP automaticamente.

    • Scegli Manualmente se il tuo Internet Provider ti ha assegnato un indirizzo IP con il tuo account. Se necessario, inserisci le informazioni che hai ricevuto dal tuo Internet Provider, ad esempio l'indirizzo IP, la maschera di sottorete, l'indirizzo router e altre informazioni.

  2. Fai clic su “Condivisione connessione” e configura la modalità utilizzata dal dispositivo per condividere la connessione Internet con i computer sul network oppure, se è connesso a un network, configura il dispositivo come bridge.

    • Scegli "Condividi un indirizzo IP (WAN) pubblico" dal menu a comparsa "Condivisione documenti" se intendi usare DHCP e NAT per condividere il tuo indirizzo IP con i computer del network.

    • Scegli "Distribuisci un intervallo di indirizzi IP" dal menu a comparsa "Condivisione documenti" se il tuo Internet Provider ti fornisce un intervallo di indirizzi IP.

    • Scegli Disattivato (modalità Bridge) se il dispositivo è collegato a un network che dispone di un server DHCP per la distribuzione degli indirizzi IP.

  3. Fai clic su Aggiorna per reinizializzare il dispositivo con queste nuove impostazioni.

Se ti colleghi a Internet utilizzando un account PPPoE (Point-to-Point Protocol over Ethernet), fai clic sul link sottostante.
