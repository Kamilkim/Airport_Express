Dirigere il traffico del network su uno specifico computer

Per garantire che le richieste vengano inviate correttamente al server web, AppleShare o FTP o a un computer specifico del network, devi definire un indirizzo IP permanente per il server o per il computer e fornire informazioni di mappatura porta in entrata alla base AirPort o a Time Capsule.

Per configurare la mappatura porta in entrata:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic su Avanzate nella barra degli strumenti, quindi su Mappatura porta.

  4. Fai clic sul pulsante Aggiungi (+) e scegli un servizio dal menu a comparsa Servizio (ad esempio, Condivisione documenti).

  5. Inserisci l'indirizzo IP del server nel campo Indirizzo IP privato.

Per utilizzare la mappatura porta, devi configurare manualmente il protocollo TCP/IP sul computer che sta utilizzando il server web, AppleShare o FTP.
