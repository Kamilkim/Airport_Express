Configurare un host di default nel network

Puoi configurare un computer del network AirPort come host di default, in modo tale che questo computer sarà esposto su Internet e riceverà tutto il traffico in entrata. Può risultare utile se utilizzi un computer sul network AirPort per giocare in rete o se desideri indirizzare tutto il traffico Internet su un solo computer.

La base o Time Capsule sono in grado di fornire gli indirizzi IP e trasmettere le informazioni Internet ai computer del network AirPort mediante DHCP (Dynamic Host Configuration Protocol) e NAT (Network Address Translation).

Per configurare un computer come host di default:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic su Internet nella barra degli strumenti, quindi su NAT.

  4. Seleziona "Abilita Host di default a" e assegna l'indirizzo IP al computer del network che desideri configurare come host di default. L'indirizzo di default è 10.0.1.25.
