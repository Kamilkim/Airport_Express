Configurare un network ospiti

Quando configuri un network ospiti, una parte della connessione a Internet è riservata agli "ospiti"; si tratta dei client wireless che possono unirsi al network ospiti e connettersi a Internet senza accedere al tuo network privato.

Se sui computer client è stata configurata la condivisione dei servizi, anche gli ospiti possono comunicare tra di loro.

Il dispositivo wireless Apple deve essere configurato per condividere il proprio indirizzo IP pubblico.

Per configurare un network ospiti:

  1. Apri Utility AirPort. Nei computer Macintosh, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo wireless sul quale intendi configurare un network ospiti, quindi fai clic su Continua.

    Usa Impostazione Assistita per configurare il dispositivo e il network privato con le impostazioni desiderate.

  3. Per default il network ospiti viene chiamato Network ospiti XXX, dove XXX è il nome del tuo network privato. Puoi cambiare il nome del network ospiti e digitare il nuovo nome nel campo "Abilita network ospiti".

  4. Assegna una password al network ospiti oppure scegli Nessuna nel menu a comparsa "Protezione network ospiti".

  5. Fai clic su Continua per terminare l'inserimento delle impostazioni nell'Impostazione Assistita, quindi fai clic su Aggiorna.

Se desideri consentire ai computer ospiti di comunicare tra di loro, controlla che sui computer client siano configurati i servizi di condivisione. Seleziona il dispositivo wireless in Utility AirPort e fai clic su "Impostazione manuale". Fai clic su AirPort nella barra degli strumenti, fai clic su "Network ospiti", quindi seleziona "Consenti ai client del network ospiti di comunicare tra loro".
