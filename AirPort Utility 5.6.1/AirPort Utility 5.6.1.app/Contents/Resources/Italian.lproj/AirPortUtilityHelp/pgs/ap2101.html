Configurare un network WPA2 Enterprise

Per configurare un network WPA2 Enterprise, è necessario configurare un server RADIUS per la gestione e l'autenticazione delle credenziali degli utenti del network, quali nomi utente, password e certificati utente. Per la configurazione del server RADIUS, consulta la documentazione fornita con il server stesso. Dopo avere configurato il server, segui le seguenti istruzioni per configurare un network WPA2 Enterprise.

Per configurare un network WPA2 Enterprise:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic su AirPort nella barra degli strumenti, quindi su Wireless.

  4. Scegli "WPA/WPA2 Enterprise" o "WPA2 Enterprise" dal menu a comparsa "Sicurezza wireless" in base alla configurazione della base e alla compatibilità dei computer client wireless connessi al network.

  5. Fai clic su "Configura RADIUS" e inserisci l'indirizzo IP, la chiave condivisa (o la password) e la porta del server di autenticazione RADIUS primario e secondario.
