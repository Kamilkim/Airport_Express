Nascondere il nome del network

Quando crei un network chiuso con una base AirPort o Time Capsule, il nome del network rimane nascosto ai computer che eseguono scansioni alla ricerca di network wireless. Gli utenti devono inserire il nome esatto del network per accedervi e può essere necessario inserire una password. Sia il nome che la password del network sono case-sensitive. Un network chiuso può garantire maggiore sicurezza.

Per creare un network chiuso:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic su AirPort nella barra degli strumenti, quindi su Wireless.

  4. Fai clic su “Opzioni network wireless”, quindi seleziona il riquadro “Crea un network chiuso”.

Per accedere a un network chiuso, scegli Altro nel menu di stato di AirPort nella barra dei menu, inserisci il nome del network e, se necessario, la password. Contatta l’amministratore del network se non conosci il nome o la password del network.
