Condividere la connessione Internet

Puoi configurare la base o Time Capsule affinché condivida un singolo indirizzo IP con tutti i client del network utilizzando DHCP (Dynamic Host Configuration Protocol) e NAT (Network Address Translation).

Se la base o Time Capsule utilizzano PPPoE per connettersi a Internet, stanno già condividendo un singolo indirizzo IP utilizzando DHCP e NAT.

Per condividere la connessione Internet:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password della base.

  3. Fai clic su Internet nella barra degli strumenti, quindi su Connessione Internet.

  4. Scegli "Condividi un indirizzo IP (WAN) pubblico" o "Distribuisci un intervallo di indirizzi IP" dal menu a comparsa Condivisione documenti.

Se abiliti DHCP e NAT condividendo di un indirizzo IP pubblico o distribuendo un intervallo di indirizzi IP, la base o Time Capsule assegnano in modo dinamico e automatico gli indirizzi IP ai computer client, semplificando la configurazione TCP/IP di ogni computer.
