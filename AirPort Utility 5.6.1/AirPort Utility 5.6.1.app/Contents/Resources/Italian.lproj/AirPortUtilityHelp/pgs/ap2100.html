Proteggere il network wireless con WPA2 (Wi-Fi Protected Access 2)

Utility AirPort supporta due modalità WPA2: la modalità Enterprise, che utilizza un server RADIUS per l’autenticazione utente, e la modalità Personal, che non richiede un server RADIUS.

La modalità Enterprise è progettata per un network di grandi dimensioni, configurato e gestito preferibilmente da un professionista IT. Per impostare un network WPA Enterprise, occorre innanzitutto configurare una connessione 802.1X. La connessione 802.1X richiede un protocollo di autenticazione, quale TTLS, LEAP o PEAP. È necessario inoltre un server di autenticazione per gestire credenziali e certificati degli utenti del network.

Per informazioni e istruzioni sulla configurazione di un network WPA Enterprise, fai clic sul link sottostante.

Configurare un network WPA2 Enterprise

La modalità Personal è progettata per gli ambienti domestici o i piccoli uffici. Può essere configurata e gestita dalla maggior parte degli utenti. La modalità Personal non richiede un server di autenticazione separato. Generalmente, per l'accesso al network, è necessario inserire un nome utente e una password.

Per configurare un network WPA2 Personal:

  1. Apri Utility AirPort. Nei computer con sistema operativo Mac OS X, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo che stai configurando, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic su AirPort nella barra degli strumenti, quindi su Wireless.

  4. Se desideri che al network accedano i computer compatibili con WPA e WPA2, scegli "WPA/WPA2 Personal" dal menu a comparsa Sicurezza wireless. Se desideri che al network accedano solo i computer compatibili con WPA2, scegli WPA2 Personal.

  5. Inserisci una password ASCII composta da un numero di caratteri ASCII compreso tra 8 a 63.
