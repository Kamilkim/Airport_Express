Configurare la base o Time Capsule per utilizzare Bonjour

Bonjour, definito talvolta come "zero-configuration networking", è una tecnologia di networking che consente l'individuazione automatica dei dispositivi senza alcuna configurazione o impostazione network.

Alcune versioni della base AirPort Extreme e tutte le versioni di Time Capsule supportano l'utilizzo di Bonjour per rendere visibili i servizi su Internet. È necessario utilizzare un nome host DNS dinamico (o un nome host globale dinamico) e la password che hai ricevuto dal tuo Internet Provider o dal tuo amministratore del network.

Quando assegni un nome host globale dinamico al tuo dispositivo wireless AirPort, puoi accedere al dispositivo al di fuori del network locale corrente, anche se l'indirizzo IP pubblico cambia ad intervalli regolari.

Quando disponi delle informazioni account, segui queste istruzioni per inserire le informazioni in Utility AirPort.

Per inserire le informazioni Dynamic DNS:

  1. Apri Utility AirPort. Nei computer Macintosh, Utility AirPort si trova nella cartella Utility della cartella Applicazioni. Mentre nei computer con sistema operativo Windows, si trova in Start > Programmi > AirPort.

  2. Seleziona il dispositivo dall'elenco, quindi fai clic su Impostazione manuale. Se necessario, inserisci la password.

  3. Fai clic sulla base o su Time Capsule, quindi fai clic su Modifica accanto al nome del campo.

  4. Seleziona “Utilizza nome host globale dinamico”, quindi inserisci le informazioni dell'account.

Contatta il tuo Internet Provider o l'amministratore del network per ulteriori informazioni sul supporto di Bonjour.
