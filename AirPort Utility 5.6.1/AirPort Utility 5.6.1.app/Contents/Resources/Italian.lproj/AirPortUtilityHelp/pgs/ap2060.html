Configurare un network AirPort

Utilizza il pannello Wireless per impostare le opzioni del network wireless.

Le opzioni wireless comprendono:

  • Configurare la modalità wireless per creare un nuovo network wireless o estenderne uno esistente.

  • Assegnazione di un nome al network. Si tratta del nome visualizzato nel menu di stato di AirPort, nella barra dei menu sui computer con sistema operativo Mac OS X. Sui computer con sistema operativo Windows, tale nome viene visualizzato sopra l'icona network wireless, nella barra di stato.

  • Impostazione della modalità radio. Scegli la frequenza del network wireless maggiormente compatibile con i computer che si connettono al network.

    • Se prevedi che si connettano al network computer che utilizzano 802.11b o 802.11g, scegli “802.11n (compatibile con 802.11b/g)” nel menu a comparsa Modalità radio.

    • Se prevedi che si connettano al network computer che utilizzano 802.11a, scegli "802.11n (compatibile con 802.11a)".

    • Se prevedi che si connettano al network solo computer che utilizzano 802.11n, scegli “Solo 802.11n (2,4 GHz)” oppure “Solo 802.11n (5 GHZ)” dal menu a comparsa “Modalità radio”.

  • Scelta di un canale per il network. Nella maggior parte dei casi, l'utilizzo dell'impostazione Automatico è la più utile, anche se possono verificarsi casi in cui sia necessario scegliere un canale specifico per il network. Se stai utilizzando Solo 802.11n, nell'intervallo di 5 gigahertz (GHz), non puoi impostare manualmente i canali.

  • Protezione del network tramite password Scegli WEP (Wired Equivalent Privacy), WPA (Wi-Fi Protected Access) o WPA2 Personal, WPA o WPA2 Enterprise oppure WEP (Transitional Security Network) in funzione della base che stai configurando e delle funzionalità dei computer client wireless che si connetteranno alla network.

  • Configurazione di ulteriori opzioni wireless. Fai clic su “Opzioni network wireless” per impostare la frequenza multicast, la potenza di trasmissione o la regione, creare un network chiuso e abilitare la resistenza alle interferenze. Se la base lo supporta e stai utilizzando la modalità radio "Solo 802.11n (5 GHz)" puoi anche selezionare l'utilizzo di canali ampi, che offrono una capacità di trasmissione dei dati nel network superiore.
