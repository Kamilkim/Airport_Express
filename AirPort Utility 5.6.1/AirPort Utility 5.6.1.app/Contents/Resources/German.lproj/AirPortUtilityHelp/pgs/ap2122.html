Steuern des Benutzerzugriffs auf Ihr Netzwerk

Sie können steuern, zu welcher Zeit und an welchen Tagen ein Computer auf Ihr AirPort Extreme-Netzwerk zugreifen kann. Sie können festlegen, wann ein Benutzer das Netzwerk verwenden darf, um auf das Internet zuzugreifen, um Dateien gemeinsam mit anderen zu nutzen oder um Spiele zu spielen.

Gehen Sie wie folgt vor, um festzulegen, wann der Zugriff auf Ihr Netzwerk erlaubt sein soll:

  1. Öffnen Sie das AirPort-Dienstprogramm, das sich auf einem Macintosh-Computer im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" befindet. Auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

  2. Wählen Sie Ihre Basisstation oder Time Capsule aus und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein.

  3. Klicken Sie in der Symbolleiste auf "AirPort" und klicken Sie dann auf "Zugriffskontrolle".

  4. Klicken Sie auf "Hinzufügen" (+) und geben Sie die MAC-Adresse und eine Beschreibung oder den Namen der Computer ein, die auf das Netzwerk zugreifen können. Sie können auch auf "Diesen Computer " klicken, um die MAC-Adresse und den Namen des verwendeten Computers hinzuzufügen.

  5. Wählen Sie einen Wert aus den Einblendmenüs aus. Wählen Sie einen Wochentag oder alle Wochentage aus dem Einblendmenü für Tage aus. Wählen Sie anschließend "ganztägig" oder "zwischen" aus dem anderen Einblendmenü aus. Bei Auswahl von "zwischen" können Sie die Tageszeiten bearbeiten, indem Sie die Zeitfelder durch Doppelklicken auswählen.

Werden keine Computer aufgelistet, können Sie das Standardverhalten für alle Benutzer anpassen, die auf das Netzwerk zugreifen wollen. Hierzu klicken Sie auf "Bearbeiten" und wählen die gewünschten Werte aus den Einblendmenüs aus.
