Einrichten eines standardmäßigen Host-Computers in Ihrem Netzwerk

Sie können einen Computer in Ihrem AirPort-Netzwerk als Standard-Host einrichten. Dieser Computer ist dann im Internet verfügbar und wickelt den gesamten eingehenden Datenverkehr ab. Ein Standard-Host ist u. U. von Nutzen, wenn Sie einen Computer in Ihrem AirPort-Netzwerk für Netzwerkspiele verwenden oder den gesamten Internet-Datenverkehr über einen einzigen Computer leiten möchten.

Ihre Basisstation oder Time Capsule muss so konfiguriert werden, dass das Gerät die Internetverbindung über DHCP (Dynamic Host Configuration Protocol) und NAT (Network Address Translation) freigibt.

Gehen Sie wie folgt vor, um einen Computer als Host-Computer einzurichten:

  1. Öffnen Sie das AirPort-Dienstprogramm. Auf einem Computer mit Mac OS X ist es im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

  2. Wählen Sie das zu konfigurierende Gerät aus und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein.

  3. Klicken Sie in der Symbolleiste auf "Internet" und klicken Sie dann auf "NAT".

  4. Wählen Sie "Standard-Host aktivieren unter" aus und weisen Sie die IP-Adresse dem Computer in Ihrem Netzwerk zu, den Sie als Standard-Host konfigurieren wollen. Die Standard-Adresse lautet 10.0.1.25.
