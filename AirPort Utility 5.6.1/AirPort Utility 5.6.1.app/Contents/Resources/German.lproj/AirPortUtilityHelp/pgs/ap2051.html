Freigeben einer USB-Festplatte in Ihrem Netzwerk

Wenn Ihre Basisstation dies unterstützt, können Sie eine USB-Festplatte an den USB-Anschluss Ihrer Basisstation anschließen und Computer in Ihrem Netzwerk (drahtlos verbunden oder verkabelt) können darauf zugreifen, um Dateien und Daten zu verwenden, bereitzustellen und zu speichern.

Schließen Sie die Festplatte am USB-Anschluss an der Rückseite der Basisstation an. Öffnen Sie das AirPort-Dienstprogramm, wählen Sie Ihre Basisstation aus und wählen Sie dann "Basisstation" > "Manuelle Konfiguration" aus. Geben Sie ggf. das Kennwort ein. Klicken Sie in der Symbolleiste auf "Laufwerke".

Hinweis: Bei Verwendung von Time Capsule muss keine externe USB-Festplatte angeschlossen werden. Time Capsule wird mit einer internen AirPort-Festplatte geliefert. Verwenden Sie die folgenden Anleitungen, um die gemeinsame Festplattennutzung für Time Capsule oder die mit Ihrer Basisstation verbundene Festplatte einzurichten.

Auf einem Computer mit Mac OS X ist das AirPort-Dienstprogramm im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

Gehen Sie wie folgt vor, um eine Festplatte für Freigabe zu konfigurieren:

  1. Klicken Sie auf "Dateifreigabe".

  2. Wählen Sie "Mit Laufwerkskennwort" oder "Mit Basisstation-Kennwort", wenn Sie die freigegebene Festplatte mit einem Kennwort schützen möchten. Wählen Sie alternativ "Mit Accounts", wenn Sie die Festplatte mithilfe von Accounts schützen möchten.

    Wenn Sie sich für die Verwendung von Accounts entscheiden, klicken Sie auf "Accounts konfigurieren". Klicken Sie dann auf "Hinzufügen" (+) und geben Sie einen Namen und ein Kennwort für jeden Benutzer ein, der auf die Festplatte zugreifen wird.

  3. Wählen Sie "Nicht zulässig", Nur lesen", "Nur schreiben" oder "Lesen und Schreiben" aus dem Einblendmenü für den AirPort-Gastzugriff, um den Zugriff durch Gäste auf dieses Volume festzulegen.

    Gastbenutzer haben Zugriff auf die Festplatte, nicht jedoch auf die Benutzer-Accounts.

  4. Aktivieren Sie das Markierungsfeld "Laufwerke über Ethernet-WAN-Anschluss freigeben", wenn Sie den Fernzugriff auf die Festplatte über den WAN-Anschluss bereitstellen möchten.

Wenn Sie einen USB-Hub an die Basisstation anschließen, können Sie mehrere Festplatten damit verbinden und über das drahtlose Netzwerk darauf zugreifen.
