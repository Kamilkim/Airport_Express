Einrichten eines Gastnetzwerks

Wenn Sie ein Gastnetzwerk einrichten, wird ein Teil Ihrer Verbindung zum Internet für "Gäste" reserviert. Drahtlose Clients können so auf das Gastnetzwerk zugreifen und eine Verbindung zum Internet herstellen, ohne auf Ihr privates Netzwerk zuzugreifen.

Wenn auf den Client-Computern Dienste zur gemeinsamen Nutzung eingerichtet wurden, können Gäste auch miteinander kommunizieren.

Das drahtlose Apple-Gerät muss für die Nutzung der öffentlichen IP-Adresse konfiguriert sein.

Gehen Sie wie folgt vor, um ein Gastnetzwerk einzurichten:

  1. Öffnen Sie das AirPort-Dienstprogramm, das sich auf einem Macintosh-Computer im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" befindet. Auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

  2. Wählen Sie das drahtlose Gerät aus, auf dem Sie ein Gastnetzwerk einrichten möchten. Klicken Sie dann auf "Fortfahren".

    Verwenden Sie den Assistenten, um die Einstellungen für Ihr Gerät und privates Netzwerk festzulegen.

  3. Standardmäßig erhält das Gastnetzwerk den Namen "XXX Gästenetzwerk", wobei XXX der Name Ihres privaten Netzwerks ist. Sie können den Namen des Gastnetzwerks ändern, indem Sie einen neuen Namen in das Feld "Gästenetzwerk aktivieren" eingeben.

  4. Legen Sie ein Kennwort für das Gastnetzwerk fest oder wählen Sie aus dem Einblendmenü "Sicherheit für Gästenetzwerk" die Option "Ohne" aus.

  5. Klicken Sie auf "Fortfahren", um weitere Einstellungen im Assistenten einzugeben. Klicken Sie dann auf "Aktualisieren".

Wenn Sie die Kommunikation von Gastcomputern untereinander erlauben möchten, stellen Sie sicher, dass auf den Client-Computern die Dienste für die Freigabe eingerichtet wurden. Wählen Sie das drahtlose Gerät im AirPort-Dienstprogramm aus und klicken Sie auf "Man. Konfiguration". Klicken Sie in der Symbolleiste auf "AirPort" und auf "Gästenetzwerk". Wählen Sie dann "Kommunikation zwischen Gästenetzwerk-Clients zulassen".
