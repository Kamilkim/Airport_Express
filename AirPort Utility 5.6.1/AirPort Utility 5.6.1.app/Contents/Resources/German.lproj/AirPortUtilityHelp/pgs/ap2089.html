Weiterleiten von Netzwerkverkehr an einen bestimmten Computer in Ihrem Netzwerk

Sie müssen eine dauerhafte IP-Adresse für den jeweiligen Server oder Computer einrichten und eingehende Informationen an die AirPort-Basisstation oder Time Capsule umleiten, um sicherzustellen, dass Anfragen korrekt an Ihren Web-, AppleShare- oder FTP-Server oder einen bestimmten Computer im Netzwerk weitergeleitet werden.

Gehen Sie wie folgt vor, um die Portumleitung einzurichten:

  1. Öffnen Sie das AirPort-Dienstprogramm. Auf einem Computer mit Mac OS X ist es im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

  2. Wählen Sie das zu konfigurierende Gerät aus und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein.

  3. Klicken Sie in der Symbolleiste auf "Erweitert" und klicken Sie dann auf "Port-Umleitung".

  4. Klicken Sie auf "Hinzufügen" (+) und wählen Sie einen Dienst aus dem Einblendmenü "Dienst" aus (etwa "Persönliche Dateifreigabe").

  5. Geben Sie die IP-Adresse des Servers in das Feld "Private IP-Adresse" ein.

Sie müssen auf dem Computer, auf dem der Web-, AppleShare- oder FTP-Server ausgeführt wird, TCP/IP manuell konfigurieren, um die Portumleitung zu verwenden.
