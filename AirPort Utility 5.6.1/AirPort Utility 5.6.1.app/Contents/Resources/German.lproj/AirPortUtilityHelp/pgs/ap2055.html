Herstellen einer Verbindung zum Internet über PPPoE

Einige Internetanbieter, die DSL- oder Kabelmodem-Verbindungen bereitstellen, verlangen, dass Sie das PPPoE-Protokoll (Point-to-Point Protocol over Ethernet) zum Herstellen einer Internetverbindung verwenden.

Wenn Sie ein DSL- oder Kabelmodem verwenden, vergewissern Sie sich, dass es mit der Telefondose oder dem Kabelanschluss sowie dem WAN-Anschluss (Wide Area Network) des drahtlosen Geräts verbunden ist, bevor Sie Ihre Basisstation konfigurieren.

Öffnen Sie das AirPort-Dienstprogramm, wählen Sie das zu konfigurierende Gerät aus und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein. Klicken Sie in der Symbolleiste auf "Internet" und klicken Sie dann auf "Internetverbindung". Wählen Sie "PPPoE" aus dem Einblendmenü "Verbinden mit" aus.

Das AirPort-Dienstprogramm befindet sich auf einem Macintosh-Computer im Ordner "Dienstprogramme" innerhalb des Ordners "Programme". Auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

Gehen Sie wie folgt vor, um Ihre Basisstation für die Verwendung von PPPoE einzurichten:

  1. Geben Sie den Benutzernamen, das Kennwort und alle weiteren Informationen ein, die Sie von Ihrem Internetanbieter erhalten haben.

    Wenn Sie Ihre Basisstation oder Time Capsule für die Verbindung via PPPoE konfigurieren, erhält das Gerät automatisch eine IP-Adresse von Ihrem Internetanbieter und gibt diese IP-Verbindung über DHCP und NAT für die Computer in Ihrem drahtlosen Netzwerk frei.

  2. Wählen Sie "Immer eingeschaltet", "Automatisch" oder "Manuell" aus dem Einblendmenü "Verbindung" aus.

    • Bei Auswahl von "Immer eingeschaltet" bleibt Ihr drahtloses Gerät so lange mit Ihrem Modem und dem Internet verbunden, wie das Modem eingeschaltet ist.

    • Wenn Sie "Automatisch" oder "Manuell" aus dem Einblendmenü "Verbindung" auswählen, müssen Sie einen Zeitwert wie "10 Minuten" aus dem Einblendmenü "Trennen nach" auswählen. Wenn Sie die Internetverbindung nicht mehr verwenden, nachdem die angegebene Zeit vergangen ist, wird die Verbindung getrennt.

    • Bei Auswahl von "Automatisch" stellt Ihr drahtloses Gerät eine Verbindung zum Modem her, das auf das Internet zugreift, wenn Sie ein Programm verwenden, das Internetzugriff erfordert (etwa ein E-Mail-, Instant-Messaging- oder Webprogramm).

    • Bei Auswahl von "Manuell" müssen Sie das Modem mit dem Internet verbinden, wenn Sie ein Programm verwenden, das Internetzugriff erfordert.

  3. Klicken Sie auf "Aktualisieren", damit das drahtlose Gerät diese neuen Einstellungen übernimmt.

Wenn Ihr Webbrowser beim Zugriff auf das Internet einen Verbindungsfehler meldet, warten Sie, bis Ihr Computer mit dem AirPort-Netzwerk verbunden ist und vergewissern Sie sich, dass das Netzwerk über Internetzugriff verfügt.

Zum Überprüfen der Verbindungen verwenden Sie das Symbol für den Status der drahtlosen Verbindung in der Menüleiste auf einem Mac und das Symbol für das drahtlose Netzwerk in der Taskleiste auf einem Windows-Computer.
