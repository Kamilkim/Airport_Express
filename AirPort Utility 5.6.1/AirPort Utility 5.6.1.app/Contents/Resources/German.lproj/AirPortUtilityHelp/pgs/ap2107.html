Einrichten von Benutzer-Accounts für mit einem AirPort-Netzwerk verbundene Festplatten

Sie können Accounts für Benutzer Ihres drahtlosen Netzwerks einrichten, um zu steuern, wie ein Benutzer an Ihre Basisstation angeschlossene USB-Festplatten nutzt. Wenn Sie Time Capsule verwenden, können Sie Accounts für Benutzer der internen AirPort-Festplatte einrichten.

Gehen Sie wie folgt vor, um einen Benutzer-Account einzurichten:

  1. Öffnen Sie das AirPort-Dienstprogramm. Auf einem Computer mit Mac OS X ist es im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

  2. Wählen Sie das zu konfigurierende Gerät aus und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein.

  3. Klicken Sie in der Symbolleiste auf "Laufwerke" und klicken Sie dann auf "Dateifreigabe".

  4. Wählen Sie "Mit Accounts" aus dem Einblendmenü "Freigegebene Laufwerke schützen" aus und klicken Sie dann auf "Accounts konfigurieren".

  5. Klicken Sie auf "Hinzufügen" (+), geben Sie den Benutzernamen ein und weisen Sie dem Benutzer dann ein Kennwort zu.

  6. Wählen Sie "Lesen und schreiben", "Nur lesen" oder "Nicht zulässig" aus dem Einblendmenü für den Dateifreigabe-Zugriff aus, je nachdem, welche Zugriffsrechte Sie dem jeweiligen Benutzer zuweisen möchten.
