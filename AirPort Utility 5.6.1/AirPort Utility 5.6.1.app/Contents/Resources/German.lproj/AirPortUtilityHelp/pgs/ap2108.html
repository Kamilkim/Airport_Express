Anschließen einer USB-Festplatte an Ihre Basisstation

Wenn Ihre Basisstation dies unterstützt, können Sie eine USB-Festplatte an den USB-Anschluss Ihrer Basisstation anschließen und Computer in Ihrem Netzwerk (drahtlos verbunden oder verkabelt) können darauf zugreifen, um Dateien und Daten zu verwenden, bereitzustellen und zu speichern.

In Time Capsule ist bereits eine AirPort-Festplatte integriert. Sie können weitere USB-Festplatten mit dem USB-Anschluss von Time Capsule verbinden.

Gehen Sie wie folgt vor, um eine USB-Festplatte an Ihre Basisstation oder Time Capsule anzuschließen:

Schließen Sie die Festplatte am USB-Anschluss an der Rückseite des Geräts an. Wenn Sie einen USB-Hub an das Gerät anschließen, können Sie mehrere Festplatten damit verbinden und über das drahtlose Netzwerk darauf zugreifen.

Das AirPort-Dienstprogramm unterstützt das Formatieren von Festplatten nicht. Formatieren Sie die Festplatte mithilfe Ihres Computers. Formatieren Sie die Festplatte mit einem Macintosh-Computer im Mac OS Extended-Format, das auch als "HFS Plus" bezeichnet wird. Wenn Sie einen Computer mit Windows verwenden, formatieren Sie sie im Format "FAT32".

Bei Verwendung einer Time Capsule haben Sie die Möglichkeit, den Inhalt der internen AirPort-Festplatte mithilfe des Bereichs "Laufwerke" des AirPort-Dienstprogramms zu löschen.

Wenn Sie Informationen über die Festplatte benötigen, öffnen Sie das AirPort-Dienstprogramm, wählen Sie Ihre Basisstation aus und wählen Sie "Basisstation" > "Manuelle Konfiguration" aus. Geben Sie ggf. das Kennwort für Ihre Basisstation ein. Klicken Sie in der Symbolleiste auf "Laufwerke".

Sie können die Zugriffsrechte für die Festplatte festlegen und im Bereich "Dateifreigabe" einen Arbeitsgruppennamen vergeben.

Öffnen Sie das AirPort-Dienstprogramm, das sich auf einem Macintosh-Computer im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" befindet. Auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".
