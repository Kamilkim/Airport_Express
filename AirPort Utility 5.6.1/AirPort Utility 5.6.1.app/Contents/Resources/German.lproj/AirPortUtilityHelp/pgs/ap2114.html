Verwenden von Profilen

Einige AirPort- und Time Capsule-Versionen können bis zu fünf Konfigurationen, so genannte "Profile", speichern. Ein Profil enthält Einstellungen der Basisstation wie Namen und Kennwörter sowie Informationen zur Netzwerkkonfiguration, etwa, wie die Basisstation die Internetverbindung herstellt.

Profile sind vor allem dann nützlich, wenn Sie Ihr Gerät häufig an verschiedenen Standorten nutzen. So können Sie zum Beispiel ein Profil für die Verwendung von Basisstation oder Time Capsule in Ihrer Wohnung einrichten, wo Sie via PPPoE auf das Internet zugreifen. Für Ihr Büro, wo Sie über DHCP auf das Internet zugreifen, verwenden Sie dann ein anderes Profil.

Wenn Sie ein neues Profil erstellen, wird eine Kopie der aktuellen Einstellungen im aktuellen Profil gespeichert. Sie können ein neues Profil hinzufügen, Profileinstellungen ändern und Ihr Gerät anschließend mit Ihren neuen Einstellungen aktualisieren.

Öffnen Sie das AirPort-Dienstprogramm, wählen Sie das Gerät aus, für das Profile hinzugefügt werden sollen, und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein.

Auf einem Computer mit Mac OS X ist das AirPort-Dienstprogramm im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

Gehen Sie wie folgt vor, um ein Profil zu erstellen oder zu bearbeiten:

  1. Wählen Sie "Profile verwalten" aus dem Menü "Basisstation" aus.

  2. Führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf einem Mac auf "Hinzufügen" (+), um die Einstellungen der aktuellen Basisstation zu kopieren. Benennen Sie die Kopie beispielsweise "Büro" und nehmen Sie die erforderlichen Änderungen vor, um Ihre Basisstation z. B. im Büro verwenden zu können.

    • Wählen Sie auf einem Windows-Computer ein Profil aus der Liste aus und klicken Sie auf "Duplizieren". Benennen Sie die Kopie beispielsweise "Büro" und klicken Sie dann auf "Fertig". Nehmen Sie die erforderlichen Änderungen vor, um Ihre Basisstation z. B. im Büro verwenden zu können.

  3. Klicken Sie auf "Aktualisieren", um Änderungen zu sichern und zu den neuen Einstellungen zu wechseln.

Wenn Sie zwischen den konfigurierten Profilen wechseln möchten, wählen Sie das gewünschte Profil aus dem Einblendmenü unten im Fenster des AirPort-Dienstprogramms aus und klicken dann auf "Aktualisieren".
