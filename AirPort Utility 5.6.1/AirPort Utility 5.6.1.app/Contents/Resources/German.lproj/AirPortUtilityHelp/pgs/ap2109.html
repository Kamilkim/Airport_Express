Umbenennen eines USB-Druckers

Sie können mit dem AirPort-Dienstprogramm den Namen eines USB-Druckers ändern, der an Ihre Basisstation oder Time Capsule angeschlossen ist.

Gehen Sie wie folgt vor, um einen USB-Drucker umzubenennen:

  1. Öffnen Sie das AirPort-Dienstprogramm. Auf einem Computer mit Mac OS X ist es im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

  2. Wählen Sie die Basisstation oder das Time Capsule-System aus, die bzw. das Sie konfigurieren, und klicken Sie dann auf "Man. Konfiguration". Geben Sie ggf. das Kennwort ein.

  3. Klicken Sie in der Symbolleiste auf "Drucker".

  4. Geben Sie einen neuen Namen in das Feld "USB-Drucker" ein.

Es ist auch möglich, einen USB-Hub mit dem USB-Anschluss der Basisstation zu verbinden und dann mehrere Drucker über den USB-Hub anzuschließen. Danach können Sie alle Drucker mit dem AirPort-Dienstprogramm umbenennen.
