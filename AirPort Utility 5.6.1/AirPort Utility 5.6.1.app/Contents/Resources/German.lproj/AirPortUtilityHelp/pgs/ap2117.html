Schützen der Informationen auf Festplatten, die mit Ihrem AirPort-Netzwerk verbunden sind

Wenn Ihre Basisstation den Anschluss an eine USB-Festplatte unterstützt oder Sie ein Time Capsule-System mit interner AirPort-Festplatte verwenden, können Sie den Zugriff auf die Festplatte steuern. Sie können den Zugriff auf die Festplatte entweder gezielt für jeden einzelnen Benutzer steuern (wenn Sie auf Ihrer Basisstation Benutzer-Accounts eingerichtet haben) oder für die Festplatte ein Kennwort festlegen.

Öffnen Sie das AirPort-Dienstprogramm, wählen Sie Ihre Basisstation oder Time Capsule aus und wählen Sie dann "Basisstation" > "Manuelle Konfiguration" aus. Geben Sie ggf. das Kennwort für das Gerät ein. Klicken Sie auf "Laufwerke".

Auf einem Computer mit Mac OS X ist das AirPort-Dienstprogramm im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" abgelegt, auf einem Windows-Computer finden Sie das Dienstprogramm unter "Start" > "Programme" > "AirPort".

Gehen Sie wie folgt vor, um eine an Ihre Basisstation angeschlossene Festplatte mit einem Kennwort zu schützen:

Wählen Sie "Mit Accounts" oder "Mit Laufwerkskennwort" aus dem Einblendmenü " Freigegebene Volumes schützen" aus.

  • Wenn Sie "Mit Laufwerkskennwort" auswählen, geben Sie ein Kennwort ein. Die Benutzer müssen dieses Kennwort eingeben, um auf die Festplatte zugreifen zu können.

  • Wenn Sie "Mit Accounts" auswählen, müssen sich die Benutzer mit dem Benutzernamen und dem Kennwort anmelden, die Sie im Bereich "Accounts" des AirPort-Dienstprogramms festgelegt haben. Klicken Sie auf den folgenden Link, um Näheres über das Konfigurieren von Benutzer-Accounts zu erfahren.

Auch der Zugriff durch Gäste auf die Festplatte kann gesteuert werden, indem Sie für Gäste Nur-Lese-Rechte, Lese- und Schreibrechte oder keine Zugriffsrechte einrichten.

Wenn Sie die Zugriffsmöglichkeiten auf die Festplatte (z. B. die Benutzer-Accounts für das Gerät oder das Festplattenkennwort) oder wenn Sie den Account-Namen geändert haben, können Benutzer möglicherweise nicht mehr auf alle Dateien auf der Festplatte zugreifen.
