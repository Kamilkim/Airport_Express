Festlegen von DHCP-Optionen für Ihre Basisstation oder Time Capsule

Wenn Sie Ihre Basisstation oder Time Capsule so eingerichtet haben, dass das Gerät DHCP (Dynamic Host Configuration Protocol) für die Bereitstellung der IP-Adressen für Computer und Geräte in Ihrem drahtlosen Netzwerk nutzt, können Sie den IP-Adressbereich auswählen, den ein Gerät verwenden soll. Außerdem können Sie festlegen, wie lange ein Computer eine gültige IP-Adresse nutzen kann, eine Begrüßungsnachricht eingeben und bestimmte IP-Adressen für bestimmte Computer reservieren, die auf das Netzwerk zugreifen.

Öffnen Sie das AirPort-Dienstprogramm (im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" auf einem Mac und unter "Start" > "Programme" > "AirPort" auf einem Windows-Computer), wählen Sie Ihr Gerät in der Liste aus und klicken Sie dann auf "Manuelle Konfiguration". Geben Sie ggf. das Kennwort ein. Klicken Sie in der Symbolleiste auf "Internet" und klicken Sie dann auf "DHCP".

Sie können folgende DHCP-Einstellungen festlegen:

  • Wählen Sie einen IP-Adressbereich aus dem Einblendmenü "DHCP-Bereich" aus. Wählen Sie "10.0", "192.168" oder "172.16" und geben Sie eine DHCP-Anfangs- und -endadresse in die zugehörigen Felder ein, abhängig davon, welche Adressen die Basisstation bereitstellen soll.

  • Geben Sie eine Zahl in das Feld "DHCP-Lease" ein und wählen Sie dann aus dem Einblendmenü die Minuten, Stunden oder Tage wunschgemäß aus.

  • Geben Sie eine Begrüßungsnachricht in das Feld "DHCP Nachricht" ein. Diese Nachricht wird angezeigt, wenn ein Computer die Verbindung zu Ihrem Netzwerk herstellt.

  • Wenn bestimmte Computer in Ihrem drahtlosen Netzwerk bestimmte IP-Adressen erhalten sollen, klicken Sie auf "Hinzufügen" (+) unter der Liste der DHCP-Reservierungen und folgen Sie den angezeigten Anleitungen, um die Reservierung zu benennen. Reservieren Sie die Adresse anhand der MAC-Adresse oder DHCP-Client-ID. Wenn Sie "MAC-Adresse" wählen, klicken Sie auf "Fortfahren" und geben Sie die MAC-Adresse und die spezifische IP-Adresse ein.

Wenn Ihr Netzwerk für die Verwendung eines LDAP-Servers (Lightweight Directory Access Protocol) konfiguriert ist, können Sie die Adresse des Servers in das Feld "LDAP-Server" eingeben, damit die Computer in Ihrem Netzwerk darauf zugreifen können.
