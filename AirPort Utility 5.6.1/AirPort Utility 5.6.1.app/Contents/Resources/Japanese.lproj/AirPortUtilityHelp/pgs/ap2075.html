インターネット接続を共有する

1 つの IP アドレスを DHCP（Dynamic Host Configuration Protocol）および NAT（Network Address Translation）を使用するネットワーク上のすべてのクライアントで共有できるように、ベースステーションまたは Time Capsule を設定することができます。

ベースステーションまたは Time Capsule が PPPoE を使ってインターネットに接続している場合は、DHCP と NAT を使って 1 つの IP アドレスを共有するようにすでに設定されています。

インターネット接続を共有するには：

  1. 「AirMac ユーティリティ」を開きます。Mac OS X を使用するコンピュータの場合は「アプリケーション」フォルダの「ユーティリティ」フォルダに、Windows コンピュータの場合は「スタート」＞「すべてのプログラム」＞「AirMac」にあります。

  2. 設定する装置を選択して、「手動設定」をクリックします。必要に応じて、ベースステーションのパスワードを入力します。

  3. ツールバーの「インターネット」をクリックしてから、「インターネット接続」をクリックします。

  4. 「接続共有」ポップアップメニューから「パブリック IP アドレスを共有」または「IP アドレスの範囲を割り当てる」を選択します。

パブリック IP アドレスを共有するか IP アドレスの範囲を割り当てる方法で DHCP と NAT を有効にすると、ベースステーションまたは Time Capsule からクライアントコンピュータに IP アドレスを動的かつ自動的に割り当てることができます。これによって、各コンピュータの TCP/IP 構成が簡単になります。
