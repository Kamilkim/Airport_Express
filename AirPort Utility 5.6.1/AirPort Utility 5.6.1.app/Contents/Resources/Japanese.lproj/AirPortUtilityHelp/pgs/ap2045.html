ワイヤレスネットワークの通信圏を拡張する

複数の AirMac Extreme または AirMac Express ベースステーションを WDS（Wireless Distribution System）として設定して、AirMac ワイヤレスネットワークの通信圏を拡張することができます。いくつかのベースステーションを WDS 方式でワイヤレス接続するときは、各ベースステーションをメインベースステーション、リモートベースステーション、またはリレーベースステーションとして設定します。

注記：802.11n ワイヤレスネットワークの通信圏を拡張する場合は、下のリンクをクリックすると分かりやすい手順が表示されます。

802.11n ネットワークの通信圏を拡張する

メインベースステーションは、インターネットに接続されているベースステーションで、その接続をリモートベースステーションおよびリレーベースステーションと共有します。リモートベースステーションは、メインベースステーションのインターネット接続をそのまま共有します。リレーベースステーションは、メインベースステーションのインターネット接続を共有し、ほかのリモートベースステーションにその接続を転送します。

いずれのベースステーション構成（メイン、リモート、およびリレー）の場合も、メインベースステーションのインターネット接続を共有するときに、クライアントコンピュータとワイヤレスで共有する方法または Ethernet 接続を使って共有する方法を利用できます。Ethernet 接続を使う場合には、クライアントコンピュータが Ethernet 経由でベースステーションに接続されている必要があります。

複数のベースステーションを WDS 方式で設定するには、各ベースステーションの AirMac ID が必要になります。AirMac ID は MAC（Media Access Controller）アドレスとも呼ばれ、AirMac Extreme ベースステーションの底面の AirMac 記号の横にあるラベル、および AirMac Express ベースステーションの電源アダプタの側面に記載されています。すべてのベースステーションを 1 つのテーブルの上に置いて、1 つの電源コンセントに接続すれば、WDS の設定を効率的に行うことができます。

WDS を設定するには、「AirMac ユーティリティ」を開き、ワイヤレス装置を選択して、「ベースステーション」＞「手動設定」と選択します。必要に応じて、パスワードを入力します。ツールバーの「AirMac」をクリックしてから、「ワイヤレス」をクリックします。

「AirMac ユーティリティ」は、Mac OS X を使用するコンピュータの場合は「アプリケーション」フォルダの「ユーティリティ」フォルダに、Windows を使用するコンピュータの場合は「スタート」＞「すべてのプログラム」＞「AirMac」にあります。

「ワイヤレス」パネルで、次の操作を行います：

  1. 「ワイヤレスのモード」ポップアップメニューから、「WDS ネットワークに接続」を選択します。

  2. 「WDS」をクリックします。

  3. 次のいずれかの操作を行います：

    • WDS メインベースステーションを設定する場合は、「追加」（＋）ボタンをクリックし、WDS リモートベースステーションの AirMac ID またはワイヤレス MAC アドレスを入力します。

    • リモートベースステーションを設定する場合は、WDS メインベースステーションの AirMac ID またはワイヤレス MAC アドレスを入力します。

    • リレーベースステーションを設定する場合は、WDS メインベースステーションの AirMac ID またはワイヤレス MAC アドレスを入力してから、「追加」（＋）をクリックして、WDS リモートベースステーションの AirMac ID またはワイヤレス MAC アドレスを入力します。

  4. WDS 内のすべてのベースステーションに同じネットワーク名とパスワードを使用します。

  5. ツールバーの「AirMac」をクリックし、「ワイヤレス」をクリックしてから、WDS 内の各ベースステーションに同じチャンネルを使用します。

デフォルトでは、「ワイヤレスクライアントを許可」チェックボックスにチェックマークが付いています。「ワイヤレスクライアントを許可」チェックボックスのチェックマークを外した場合、後でこのベースステーションの設定を変更したいときには、Ethernet ケーブルでベースステーションの LAN ポートに接続する必要があります。ベースステーションにワイヤレス接続するには、ベースステーションの構成を変更する必要があります。

Ethernet を使ってベースステーションが接続されている場合は、「単一の Ethernet ネットワークで複数のベースステーションを使用する」を参照してワイヤレスネットワークの通信圏を拡張してください。
