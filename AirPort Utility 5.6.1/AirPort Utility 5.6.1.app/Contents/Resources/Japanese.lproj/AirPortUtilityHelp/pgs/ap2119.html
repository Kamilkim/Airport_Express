ベースステーションまたは Time Capsule の DHCP オプションを設定する

DHCP（Dynamic Host Configuration Protocol）を使ってワイヤレスネットワーク上のコンピュータや装置に IP アドレスを割り当てるようにベースステーションまたは Time Capsule を設定している場合は、装置が使用する IP アドレスの範囲を選択したり、コンピュータに割り当てる IP アドレスの有効期間を設定したり、ウェルカムメッセージを入力したり、ネットワークに接続する特定のコンピュータに割り当てるための IP アドレスを予約したりできます。

「AirMac ユーティリティ」（Mac の場合は「アプリケーション」フォルダの「ユーティリティ」フォルダに、Windows コンピュータの場合は「スタート」＞「すべてのプログラム」＞「AirMac」にあります）を開き、リストから装置を選択して、「手動設定」をクリックします。必要に応じて、パスワードを入力します。ツールバーの「インターネット」をクリックしてから、「DHCP」をクリックします。

次の DHCP オプションを設定できます：

  • 「DHCP の範囲」ポップアップメニューから、IP アドレスの範囲を選択します。ベースステーションから割り当てたいアドレスに応じて「10.0」、「192.168」、または「172.16」を選択してから、「DHCP の開始アドレス」フィールドと「DHCP の終了アドレス」フィールドに開始アドレスと終了アドレスを入力します。

  • 「DHCP リース」フィールドに数字を入力し、ポップアップメニューから「分」、「時間」、または「日」を選択します。

  • 「DHCP メッセージ」フィールドにウェルカムメッセージを入力します。このメッセージは、コンピュータがネットワークに接続したときに表示されます。

  • ワイヤレスネットワーク上の特定のコンピュータに特定の IP アドレスを割り当てるには、「DHCP の予約」リストの下にある「追加」（＋）をクリックしてから、画面に表示される指示に従って予約の名前を入力します。MAC アドレスまたは DHCP クライアント ID によってアドレスを予約します。MAC アドレスを選択した場合は、「続ける」をクリックし、MAC アドレスと特定の IP アドレスを入力します。

LDAP（Lightweight Directory Access Protocol）サーバを使用するようにネットワークが設定されている場合は、「LDAP サーバ」フィールドにサーバのアドレスを入力すると、ネットワーク上のコンピュータからサーバにアクセスできるようになります。
