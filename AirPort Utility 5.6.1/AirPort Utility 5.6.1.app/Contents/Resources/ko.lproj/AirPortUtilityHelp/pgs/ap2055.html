PPPoE를 사용하여 인터넷에 연결하기

일부 DSL 및 케이블 모뎀 인터넷 서비스 제공업체는 인터넷 연결에 PPPoE(Point-to-Point Protocol over Ethernet)의 사용을 요구할 수 있습니다.

DSL 또는 케이블 모뎀을 사용한다면, 베이스 스테이션을 설정하기 전에 해당 장비가 전화나 케이블 잭, 그리고 해당 무선 장비의 WAN(Wide Area Network) 포트에 연결되었는지 확인하십시오.

AirPort 유틸리티를 열고 설정하려는 장비를 선택한 다음, 수동 설정을 클릭하십시오. 필요하다면 암호를 입력하십시오. 도구 막대에서 인터넷을 클릭하고 인터넷 연결을 클릭하십시오. 연결 팝업 메뉴에서 PPPoE를 선택하십시오.

AirPort 유틸리티는 Mac에서는 응용 프로그램 폴더 안의 유틸리티 폴더, Windows 컴퓨터에서는 시작 > 모든 프로그램 > AirPort에 있습니다.

PPPoE를 사용하도록 베이스 스테이션을 설정하려면,

  1. 사용자 이름, 암호 및 해당 인터넷 서비스 제공업체로부터 받은 기타 모든 정보를 입력하십시오.

    PPPoE를 사용하여 연결하도록 베이스 스테이션이나 Time Capsule을 설정하면, 장비가 해당 ISP로부터 자동으로 IP 주소를 받고 DHCP 및 NAT를 사용하여 무선 네트워크상에 있는 컴퓨터들과 해당 IP 연결을 공유합니다.

  2. 연결 팝업 메뉴에서 항상 켬, 자동 또는 수동을 선택하십시오.

    • 항상 켬을 선택하면, 무선 장비가 모뎀이 켜져 있는 동안 항상 모뎀과 인터넷에 연결된 상태를 유지합니다.

    • 연결 팝업 메뉴에서 자동 또는 수동을 선택하면 “대기시 연결 해제” 팝업 메뉴에서 “10분”과 같은 특정 시간을 선택해야 합니다. 인터넷 응용 프로그램이 인터넷을 사용하지 않는다면 특정 시간 후 인터넷에서 연결 해제됩니다.

    • 자동을 선택하면 해당 무선 장비가 모뎀에 연결되며, 이 모뎀은 이메일, 인스턴트 메시지 또는 웹 응용 프로그램과 같이 인터넷 연결이 필요한 응용 프로그램을 사용할 때 인터넷에 연결됩니다.

    • 수동을 선택하면 인터넷 연결이 필요한 응용 프로그램을 사용할 때 모뎀을 인터넷에 연결해야 합니다.

  3. 업데이트를 클릭하여 이들 새로운 설정으로 무선 장비를 재설정하십시오.

인터넷에 접속할 때 웹 브라우저에서 연결 오류가 보고된다면, 컴퓨터가 AirPort 네트워크에 연결되고 네트워크가 인터넷에 연결될 때까지 기다리십시오.

연결을 확인하려면 Mac에서는 메뉴 막대에 있는 무선 연결 상태 아이콘을, Windows 컴퓨터에서는 상태 표시줄에 있는 무선 네트워크 아이콘을 사용하십시오.
