인터넷 연결 공유하기

DHCP(Dynamic Host Configuration Protocol) 및 NAT(Network Address Translation)를 사용하여 네트워크상의 모든 클라이언트와 단일 IP 주소를 공유하도록 베이스 스테이션이나 Time Capsule을 설정할 수 있습니다.

베이스 스테이션이나 Time Capsule에서 PPPoE를 사용하여 인터넷에 연결한다면, 이미 DHCP 및 NAT을 사용하여 단일 IP 주소를 공유하고 있는 것입니다.

인터넷 연결을 공유하려면,

  1. Mac OS X을 사용하는 컴퓨터에서는 응용 프로그램 폴더 안에 있는 유틸리티 폴더, Windows 컴퓨터에서는 시작 > 모든 프로그램 > AirPort에 있는 AirPort 유틸리티를 여십시오.

  2. 설정할 장비를 선택하고 수동 설정을 클릭하십시오. 필요하다면 베이스 스테이션 암호를 입력하십시오.

  3. 도구 막대에서 인터넷을 클릭하고 인터넷 연결을 클릭하십시오.

  4. 연결 공유 팝업 메뉴에서 "공용 IP 주소 공유" 또는 "IP 주소 범위 분배"를 선택하십시오.

공용 IP 주소를 공유하여 DHCP 및 NAT을 활성화하거나 IP 주소 범위를 분배하면 베이스 스테이션 또는 Time Capsule이 클라이언트 컴퓨터에 IP 주소를 동적이고 자동으로 지정하므로, 각 컴퓨터의 TCP/IP 구성이 간소화됩니다.
