이더넷을 사용하여 인터넷에 연결하기

DSL 또는 케이블 모뎀을 사용하여 인터넷에 연결하거나 LAN(Local Area Network)을 사용하여 인터넷에 연결한다면, 베이스 스테이션 또는 Time Capsule을 해당 모뎀 또는 네트워크에 연결하고 무선 네트워크상에 있는 컴퓨터 및 클라이언트와 인터넷 연결을 공유할 수 있습니다.

베이스 스테이션에서 또는 Time Capsule이 ISP(인터넷 서비스 제공업체) 또는 네트워크로부터 인터넷 주소를 받는 방식과 장비가 해당 네트워크상에 있는 컴퓨터들과 인터넷 연결을 공유하는 방식을 설정해야 합니다.

AirPort 유틸리티를 열고 설정하려는 장비를 선택한 다음, 수동 설정을 클릭하십시오. 필요하다면 암호를 입력하십시오. 도구 막대에서 인터넷을 클릭하고 인터넷 연결을 클릭하십시오. 연결 팝업 메뉴에서 이더넷을 선택한 다음 TCP/IP를 클릭하십시오.

AirPort 유틸리티는 Mac OS X을 사용하는 컴퓨터에서는 응용 프로그램 폴더 안에 있는 유틸리티 폴더에, Windows 컴퓨터에서는 시작 > 모든 프로그램 > AirPort에 있습니다.

이더넷을 사용하여 연결하도록 장비를 설정하려면,

  1. IPv4 구성 팝업 메뉴에서 다음 중 하나를 선택하여 무선 장비에서 ISP로부터 IP 주소를 수신하는 방법을 설정하십시오.

    • ISP에서 IP 주소를 자동으로 보낸다면 DHCP 사용을 선택하십시오.

    • ISP에서 계정과 함께 IP 주소를 제공하였다면 수동을 선택하십시오. IP 주소, 서브네트 마스크, 라우터 주소 및 적절한 기타 정보와 같은 ISP에서 받은 정보를 입력하십시오.

  2. 연결 공유를 클릭하고 장비에서 네트워크상에 있는 컴퓨터와 인터넷 연결을 공유하는 방법을 설정하거나, 장비가 네트워크에 연결되어 있다면 브리지로 설정하십시오.

    • DHCP 및 NAT를 사용하여 네트워크상에 있는 컴퓨터들과 IP 주소를 사용하려면 연결 공유 팝업 메뉴에서 "공용 IP 주소 공유"를 선택하십시오.

    • ISP에서 특정 범위의 IP 주소를 제공한다면 연결 공유 팝업 메뉴에서 "IP 주소 범위 분배"를 선택하십시오.

    • 장비가 IP 주소들을 분배하는 DHCP 서버가 있는 네트워크에 연결되어 있다면 끔(브리지 모드)을 선택하십시오.

  3. 업데이트를 클릭하여 이들 새로운 설정으로 장비를 재설정하십시오.

PPPoE(Point-to-Point Protocol over Ethernet) 계정을 사용하여 인터넷에 연결한다면 아래 링크를 클릭하십시오.
