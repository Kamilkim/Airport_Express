컴퓨터에서 유효한 IP 주소를 보유하는 시간 조절하기

AirPort 네트워크 내에서 컴퓨터 한대당 IP 주소가 유효한 시간을 조절하려면, DHCP(Dynamic Host Configuration Protocol) 임대 시간을 설정하십시오.

DHCP 임대 시간을 설정하려면,

  1. Mac OS X을 사용하는 컴퓨터에서는 응용 프로그램 폴더 안에 있는 유틸리티 폴더, Windows 컴퓨터에서는 시작 > 모든 프로그램 > AirPort에 있는 AirPort 유틸리티를 여십시오.

  2. 설정할 장비를 선택하고 수동 설정을 클릭하십시오. 필요하다면 암호를 입력하십시오.

  3. 도구 막대에서 인터넷을 클릭하고 연결 공유 팝업 메뉴에서 “공용 IP 주소 공유” 또는 “IP 주소 범위 분배”가 선택되었는지 확인한 다음 DHCP를 클릭하십시오.

  4. DHCP 임대란에 숫자를 입력하고 팝업 메뉴에서 분, 시 또는 일을 선택하십시오.

아주 짧은 임대를 사용하여, DHCP는 사용할 수 있는 IP 주소보다 컴퓨터가 많은 네트워크를 동적으로 재구성할 수 있습니다.

DHCP 예약을 설정했다면 목록에 있는 컴퓨터는 DHCP 임대가 만료되면 새로운 DHCP 주소를 가져올 수 없습니다.
