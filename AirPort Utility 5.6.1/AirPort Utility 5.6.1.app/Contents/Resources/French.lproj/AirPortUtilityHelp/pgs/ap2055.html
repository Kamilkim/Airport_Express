Connexion à Internet via PPPoE

Certains fournisseurs d’accès à Internet par modem DSL et câble exigent que vous utilisiez le protocole PPPoE (Point-to-Point Protocol over Ethernet) pour vous connecter à Internet.

Si vous utilisez un modem DSL ou câble, vérifiez qu’il est branché sur la prise du téléphone ou du câble d’une part et sur le port de réseau étendu (WAN) de votre périphérique sans fil d’autre part, avant de configurer votre borne d’accès.

Ouvrez Utilitaire AirPort, sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe. Cliquez sur Internet dans la barre d’outils, puis sur Connexion à Internet. Choisissez PPPoE dans le menu local Connexion via.

Utilitaire AirPort se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac, ou dans Démarrer > Tous les programmes > AirPort dans le cas d’un PC fonctionnant sous Windows.

Pour configurer votre borne d'accès de façon à utiliser PPPoE :

  1. Saisissez le nom d’utilisateur, le mot de passe et toute autre information nécessaire fournie par votre fournisseur d’accès à Internet.

    Lorsque vous configurez votre borne d’accès ou votre disque Time Capsule de façon à vous connecter à l’aide de PPPoE, le périphérique reçoit automatiquement une adresse IP de la part de votre FAI et utilise DHCP et NAT pour partager cette connexion IP avec les ordinateurs placés sur votre réseau sans fil.

  2. Choisissez Toujours active, Automatique ou Manuelle dans le menu local Connexion.

    • Si vous choisissez Toujours active, votre périphérique reste connecté au modem et à Internet aussi longtemps que le modem est allumé.

    • Si vous choisissez Automatique ou Manuelle dans le menu local Connexion, vous devez choisir un incrément tel que « 10 minutes » dans le menu local Déconnecter si inactif. Si le délai est écoulé alors que vous n’utilisez pas d’application Internet, vous êtes déconnecté d’Internet.

    • Si vous choisissez Automatique, votre périphérique sans fil établit une connexion avec le modem qui permet à une application, telle qu’une application de messagerie électronique, de messagerie instantanée ou web, d’accéder à Internet.

    • Si vous choisissez Manuelle, vous devez vous assurer que le modem se connecte à Internet lorsque vous utilisez une application qui transmet des données via Internet.

  3. Cliquez sur Mettre à jour pour réinitialiser le périphérique sans fil avec ces nouveaux réglages.

Si votre navigateur web signale une erreur de connexion lorsque vous accédez à Internet, attendez que votre ordinateur soit connecté au réseau AirPort et assurez-vous que ce dernier est connecté à Internet.

Pour vérifier les connexions, utilisez l’icône d’état de la connexion sans fil située dans la barre des menus sur un ordinateur Mac, ou l’icône de réseau sans fil située dans la barre d’état d’un PC sous Windows.
