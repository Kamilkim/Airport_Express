Désactivation du protocole SNMP

Les réseaux gérés par SNMP (Simple Network Management Protocol ou « Protocole simple de gestion de réseau » en français) peuvent subir des attaques de « déni de service ». (Par défaut, le protocole SNMP est activé dans Utilitaire AirPort.) De même, si vous autorisez votre borne d’accès à être configurée à distance via le port WAN, vous courez le risque que des utilisateurs non autorisés modifient vos réglages réseau.

Pour contribuer à protéger votre réseau et vos périphériques sans fil, désactivez le protocole SNMP.

Pour désactiver SNMP sur votre borne d’accès :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Avancé dans la barre d’outils, puis sur « Journalisation et Statistiques ».

  4. Désactivez la case à cocher Autoriser SNMP.

Vous pouvez choisir d’autoriser le protocole SNMP sur le port WAN en cochant la case « Autoriser SNMP via WAN ». Cette option vous permet de gérer vos périphériques à distance.

Cliquez sur le lien « Journalisation des informations du réseau sans fil » ci-dessous pour savoir comment configurer le périphérique sans fil pour consigner les informations concernant le réseau.
