Si vous oubliez le mot de passe de votre borne d’accès, de votre disque Time Capsule ou de votre réseau

Si vous oubliez le mot de passe de votre borne d’accès, de votre disque Time Capsule ou de votre réseau AirPort, vous pouvez réinitialiser le périphérique puis sélectionner un nouveau mot de passe.

Pour réinitialiser le mot de passe :

  1. Lorsque le périphérique est branché, utilisez la pointe d’un crayon pour appuyer sur le bouton de réinitialisation et maintenez-le enfoncé pendant quelques secondes. Les mots de passe du périphérique et du réseau sont temporairement réinitialisés sur « public ».

  2. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  3. Sélectionnez le périphérique que vous avez réinitialisé, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe. Le mot de passe par défaut est « public ».

  4. Cliquez sur AirPort dans la barre d’outils, puis sur Borne d’accès.

  5. Attribuez un nouveau mot de passe au périphérique, puis cliquez sur Mettre à jour.

Si vous ne reconfigurez pas le périphérique dans les cinq minutes imparties après la réinitialisation, vous devrez à nouveau le réinitialiser.
