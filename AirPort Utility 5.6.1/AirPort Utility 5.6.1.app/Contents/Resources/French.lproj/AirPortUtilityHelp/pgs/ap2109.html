Modification du nom d’une imprimante USB

Vous pouvez utiliser Utilitaire AirPort pour modifier le nom de l’imprimante USB connectée à votre borne d’accès ou à votre disque Time Capsule.

Pour modifier le nom d’une imprimante USB :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez la borne d’accès ou le disque Time Capsule que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Imprimantes dans la barre d’outils.

  4. Tapez un nouveau nom dans le champ Imprimantes USB.

Vous pouvez également brancher un concentrateur USB sur le port USB de la borne d’accès, puis plusieurs imprimantes sur le concentrateur USB. Utilitaire AirPort peut vous permettre ensuite de renommer toutes les imprimantes.
