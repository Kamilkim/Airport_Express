Réglage des options DHCP de votre borne d’accès ou votre disque Time Capsule

Si vous avez configuré votre borne d’accès ou votre disque Time Capsule pour utiliser le protocole DHCP (Dynamic Host Configuration Protocol) pour l’attribution d’adresses IP aux ordinateurs et aux périphériques de votre réseau sans fil, vous pouvez choisir la plage d’adresses IP qu’un périphérique utilise, déterminer la durée de validité d’une adresse IP sur un ordinateur, saisir un message de bienvenue et réserver des adresses IP en vue de les attribuer à des ordinateurs spécifiques qui accèdent à votre réseau.

Ouvrez Utilitaire AirPort (situé dans le dossier Utilitaires du dossier Applications sur un Mac et dans Démarrer > Tous les programmes > AirPort sur un ordinateur Windows), sélectionnez votre périphérique dans la liste, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe. Cliquez sur Internet dans la barre d’outils, puis sur DHCP.

Vous pouvez définir les options DHCP suivantes :

  • Sélectionnez une plage d’adresses IP dans le menu local « Plage d’adresses DHCP ». Choisissez entre 10.0, 192.168 et 172.16, puis tapez une adresse de début et une adresse de fin dans les champs Première adresse DHCP et Dernière adresse DHCP, en fonction des adresses que la borne d’accès doit attribuer.

  • Saisissez un nombre dans le champ Bail DHCP, puis choisissez le nombre de minutes, d’heures ou de jours dans le menu local.

  • Tapez un message de bienvenue dans le champ Message DHCP. Ce message s’affiche lorsqu’un ordinateur accède à votre réseau.

  • Pour fournir des adresses IP spécifiques à certains ordinateurs de votre réseau sans fil, cliquez sur Ajouter (+) sous la liste Réservations DHCP, puis suivez les instructions à l’écran pour nommer la réservation. Réservez l’adresse à partir de l’adresse MAC ou de l’identifiant client DHCP. Si vous choisissez Adresse MAC, cliquez sur Continuer, puis tapez l’adresse MAC et l’adresse IP spécifique.

Si votre réseau est configuré pour exploiter un serveur LDAP (Lightweight Directory Access Protocol), vous pouvez taper l’adresse de ce serveur dans le champ Serveur LDAP ; les ordinateurs sur votre réseau y auront alors accès.
