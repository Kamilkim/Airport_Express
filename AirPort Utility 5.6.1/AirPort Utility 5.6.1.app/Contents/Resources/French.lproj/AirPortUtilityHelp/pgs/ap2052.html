Mise à jour de votre borne d’accès ou de votre disque Time Capsule

Vous pouvez mettre à niveau le programme interne de votre borne d’accès AirPort ou de votre disque Time Capsule lorsqu’un nouveau programme est mis à disposition. Vous serez averti de la présence d’un nouveau programme interne à la prochaine ouverture d’Utilitaire AirPort. Suivez les instructions à l’écran pour télécharger et installer le nouveau programme interne.

Vous pouvez également rechercher des mises à jour manuellement.

Pour mettre à jour le programme interne manuellement :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Choisissez « Rechercher les mises à jour » dans le menu Utilitaire AirPort.

  3. Cliquez sur Télécharger pour télécharger toutes les mises à jour de programme interne disponibles ou sur Afficher les détails pour sélectionner un programme interne particulier pour le périphérique sans fil que vous mettez à jour.

  4. Une fois le programme interne téléchargé, cliquez sur Mettre à jour pour l’installer sur votre périphérique. Si vous mettez à jour plusieurs périphériques, cliquez sur Afficher les détails afin de sélectionner ceux que vous souhaitez mettre à jour.

Si vous avez téléchargé un programme interne mais ne l’avez pas installé, vous pouvez y procéder plus tard. Ouvrez Utilitaire AirPort, sélectionnez le périphérique à mettre à jour, puis cliquez sur Configuration manuelle. Cliquez sur Mettre à jour dans la sous-fenêtre Résumé des réglages AirPort pour installer le programme interne.
