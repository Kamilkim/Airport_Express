Configuration d’un hôte par défaut sur votre réseau

Vous pouvez configurer comme hôte par défaut un ordinateur de votre réseau AirPort afin qu’il soit accessible d’Internet et se charge de tout le trafic entrant. Cela peut s’avérer utile si vous utilisez un ordinateur de votre réseau AirPort pour jouer en réseau ou que vous souhaitez acheminer tout le trafic Internet à travers un seul ordinateur.

Votre borne d’accès ou disque Time Capsule doit être configuré(e) pour partager sa connexion à Internet via les protocoles DHCP (Dynamic Host Configuration Protocol) et NAT (Network Address Translation, traduction d’adresses réseau).

Pour configurer un ordinateur en tant qu'hôte par défaut :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Internet dans la barre d’outils, puis sur NAT.

  4. Sélectionnez « Activer l’hôte par défaut sur » et attribuez une adresse IP à l’ordinateur sur votre réseau que vous voulez configurer comme hôte par défaut. L’adresse par défaut est 10.0.1.25.
