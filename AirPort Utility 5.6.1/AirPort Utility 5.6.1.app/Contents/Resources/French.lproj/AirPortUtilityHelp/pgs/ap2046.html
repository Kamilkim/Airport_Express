Contrôle de l’accès à votre réseau sans fil

Grâce au contrôle d’accès, vous pouvez indiquer quels ordinateurs peuvent envoyer ou recevoir des informations via le réseau sans fil. Si votre périphérique sans fil le permet, vous pouvez aussi contrôler les heures et les jours auxquels un ordinateur peut accéder au réseau.

Ce contrôle permet de refuser l’accès au réseau AirPort aux ordinateurs qui ne figurent pas sur la liste de contrôle d’accès.

Utilitaire AirPort est prévu pour régler les options d’accès. Ouvrez Utilitaire AirPort, sélectionnez votre périphérique sans fil, puis choisissez Borne d’accès > Configuration manuelle. Le cas échéant, saisissez le mot de passe. Cliquez sur AirPort dans la barre d’outils, puis sur Contrôle d’accès.

Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

Pour configurer le contrôle d’accès :

Sélectionnez Local (proposé sur certains modèles de la borne d’accès AirPort), Accès programmé ou RADIUS dans le menu local « Contrôle d’accès d’adresses MAC », selon le périphérique sans fil que vous configurez.

  • Si vous avez choisi Local, cliquez sur le bouton Ajouter (+) et tapez l’identifiant AirPort ou l’adresse MAC sans fil du client auquel vous voulez accorder l’accès au réseau, ou cliquez sur Cet ordinateur pour ajouter l’identifiant AirPort de l’ordinateur que vous utilisez. Utilisez le champ Description pour saisir le nom de l’ordinateur dans la liste.

  • Si vous avez choisi Accès programmé, cliquez sur le bouton Ajouter (+) et tapez l’adresse MAC et la description ou le nom des ordinateurs auxquels vous voulez donner l’accès au réseau. Vous pouvez aussi cliquer sur Cet ordinateur pour ajouter l’adresse MAC et le nom de l’ordinateur que vous utilisez pour configurer la borne d’accès. Choisissez une valeur dans les menus locaux. Choisissez un jour de la semaine ou Tous les jours dans le menu local des jours, puis choisissez soit « toute la journée » soit « entre » dans le menu local annexe. Si vous avez choisi « entre », vous pouvez modifier les heures du jour en double-cliquant dans les champs des heures.

  • Si vous avez choisi RADIUS, saisissez son type de service, ses adresses IP, le secret partagé et le port principal du serveur RADIUS principal. Saisissez les informations relatives au serveur RADIUS secondaire, s’il y en a un. Consultez l’administrateur du serveur si vous ne disposez pas de ces informations.

Si vous utilisez le contrôle d’accès local dans un système WDS (Wireless Distribution System), copiez la liste de contrôle d’accès sur toutes les bornes d’accès du réseau.
