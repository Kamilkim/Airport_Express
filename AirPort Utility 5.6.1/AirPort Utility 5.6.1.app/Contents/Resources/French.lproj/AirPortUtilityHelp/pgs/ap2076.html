Configuration de l’impression à distance

Si votre borne d’accès ou votre disque Time Capsule est connecté à un réseau Ethernet et qu’une imprimante USB compatible y est également connectée, les ordinateurs clients du réseau Ethernet peuvent imprimer sur cette imprimante. Cela s’appelle l’impression à distance.

Pour activer l’impression à distance :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Imprimantes dans la barre d’outils et assurez-vous que l’imprimante apparaît dans la liste.

Sélectionner « Partager les imprimantes sur WAN » pour partager l’imprimante sur le port WAN du périphérique sans fil. Vous pouvez également partager l’imprimante sur Internet à l’aide de Bonjour.
