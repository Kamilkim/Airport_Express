Utilisation de profils

Certaines versions de la borne d’accès AirPort et du disque Time Capsule permettent de stocker jusqu’à cinq configurations, également appelées profils. Un profil contient les réglages de la borne d’accès, comme des noms et des mots de passe, ainsi que des informations sur la configuration réseau, telles que le mode de connexion à Internet via la borne d’accès.

L’utilisation de profils peut être utile si vous déplacez votre périphérique d’un endroit à l’autre. Vous pourriez, par exemple, disposer d’un profil pour utiliser votre borne d’accès ou votre disque Time Capsule à la maison, où vous vous connectez à Internet via PPPoE. Vous pourriez disposer d’un autre profil pour le bureau, où vous vous connectez à Internet via DHCP.

Lorsque vous créez un profil, une copie des réglages actuels est stockée dans le profil en cours. Vous pouvez ajouter un nouveau profil, en modifier les réglages, puis mettre à jour votre périphérique vers ces nouveaux réglages.

Ouvrez Utilitaire AirPort, sélectionnez le périphérique auquel vous voulez ajouter des profils, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

Pour créer ou modifier un profil :

  1. Choisissez Gérer les profils dans le menu Borne d’accès.

  2. Effectuez l’une des opérations suivantes :

    • Sur un ordinateur Macintosh, cliquez sur Ajouter (+) pour copier les réglages de la borne d’accès. Attribuez un nom à la copie, par exemple « Bureau », et effectuez les modifications appropriées à la borne d’accès en fonction de l’usage que vous en faites.

    • Sur un ordinateur fonctionnant sous Windows, sélectionnez un profil de la liste puis cliquez sur Dupliquer. Attribuez un nom à la copie, par exemple « Bureau », puis cliquez sur Terminé. Apportez les modifications nécessaires à votre borne d’accès, selon l’usage que vous en faites.

  3. Cliquez sur « Mettre à jour » pour enregistrer les changements et activer les nouveaux réglages.

Pour passer d’un profil configuré à un autre, choisissez celui à utiliser dans le menu local Profils placé en bas de la fenêtre d’Utilitaire AirPort, puis cliquez sur « Mettre à jour ».
