Attribution d’une adresse DHCP spécifique à des ordinateurs clients

Vous pouvez configurer une borne d’accès 802.11n ou un disque Time Capsule en vue de réserver une adresse DHCP pour des ordinateurs clients de votre réseau AirPort. Lorsque ces ordinateurs clients accèdent à votre réseau, votre borne d’accès ou votre disque Time Capsule leur attribue la même adresse DHCP qu’auparavant.

Pour que vous puissiez procéder à une réservation d’adresse DHCP, votre borne d’accès ou votre disque Time Capsule doit être configuré(e) pour attribuer des adresses IP aux ordinateurs clients de votre réseau via DHCP.

Pour configurer des réservations d’adresses DHCP :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Internet dans la barre d’outils, puis sur DHCP.

  4. Cliquez sur Ajouter (+) sous la liste Réservations DHCP et suivez les instructions à l’écran pour réserver une adresse DHCP. Vous devez connaître l’adresse MAC de l’ordinateur client pour lequel vous réservez l’adresse ; vous devez aussi décider de l’adresse à réserver dans la plage d’adresses DHCP.
