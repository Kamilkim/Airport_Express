Partage d’un disque dur USB sur votre réseau

Si votre borne d’accès le permet, vous pouvez connecter un disque dur USB au port USB de votre borne d’accès afin que les ordinateurs de votre réseau (qu’ils soient câblés ou sans fil) puissent s’y connecter pour accéder à des fichiers et des données, les partager et les stocker.

Branchez le disque dur sur le port USB à l’arrière de la borne d’accès. Ouvrez Utilitaire AirPort, sélectionnez votre borne d’accès, puis choisissez Borne d’accès > Configuration manuelle. Le cas échéant, saisissez le mot de passe. Cliquez sur Disques dans la barre d’outils.

Remarque : si vous utilisez un disque Time Capsule, vous n’avez pas besoin de connecter un disque dur USB externe. Time Capsule est équipé d’un disque AirPort interne. Utilisez les instructions suivantes pour configurer le partage de disque sur votre disque Time Capsule ou sur le disque dur connecté à votre borne d’accès.

Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

Pour configurer le partage de disques :

  1. Cliquez sur Partage de fichiers.

  2. Choisissez « Avec un mot de passe de disque » ou « Avec le mot de passe de la borne » pour sécuriser le disque partagé avec un mot de passe ou « Avec les comptes » pour sécuriser le disque à l’aide de comptes.

    Si vous avez choisi d’utiliser des comptes, cliquez sur Configurer les comptes et sur le bouton Ajouter (+), puis saisissez un nom et un mot de passe pour chacun des utilisateurs devant accéder au disque.

  3. Dans le menu local « Accès invité aux disques AirPort », choisissez parmi les valeurs « Non autorisé », « Lecture seule » ou « Lecture et écriture » pour attribuer l’accès invité au disque.

    Les invités obtiennent alors l’accès au disque dur, mais pas les comptes utilisateur définis.

  4. Cochez la case « Partager des disques via le port WAN Ethernet » si vous souhaitez accorder un accès à distance au disque via le port WAN.

Si vous branchez un concentrateur USB sur la borne d’accès, vous pouvez y connecter plusieurs disques durs et y accéder à partir du réseau sans fil.
