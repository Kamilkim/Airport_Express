Extension de la portée d’un réseau 802.11n

Utilitaire AirPort permet d’étendre facilement la portée d’un réseau 802.11n si vous connectez une autre borne d’accès 802.11n ou un disque Time Capsule. La connexion de deux périphériques 802.11n simplifie grandement le processus de configuration WDS.

Pour amplifier la portée d’un réseau 802.11n :

  1. Ouvrez Utilitaire AirPort et sélectionnez la borne d’accès ou le disque Time Capsule qui se connectera à Internet.

  2. Cliquez sur Configuration manuelle. Si nécessaire, tapez le mot de passe de la borne d’accès.

  3. Cliquez sur AirPort dans la barre d’outils, puis sur Sans fil.

  4. Sélectionnez « Créer un réseau sans fil » dans le menu local Mode sans fil puis cochez la case « Autoriser les extensions à ce réseau ».

  5. Sélectionnez ensuite le périphérique qui va étendre ce réseau, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  6. Choisissez « Étendre un réseau sans fil » dans le menu local Mode sans fil puis choisissez le réseau que vous désirez étendre via le menu local Nom de réseau.

  7. Le cas échéant, saisissez le mot de passe du réseau et du périphérique.

Si vos bornes d’accès sont connectées via Ethernet, consultez la section «  Utilisation de plusieurs bornes d’accès sur un réseau Ethernet » pour étendre la portée du réseau sans fil.

Si vous étendez la portée d’un réseau sans fil utilisant la norme 802.11g ou 11b, vérifiez le site web d’assistance Apple.
