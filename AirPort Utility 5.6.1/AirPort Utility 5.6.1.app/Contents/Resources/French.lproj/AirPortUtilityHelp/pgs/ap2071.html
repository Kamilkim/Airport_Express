Masquage du nom de votre réseau

Lorsque vous créez un réseau fermé avec une borne d’accès AirPort ou un disque Time Capsule, le nom du réseau demeure masqué pour les ordinateurs effectuant une recherche des réseaux sans fil. Les utilisateurs doivent taper le nom exact du réseau pour se connecter au réseau AirPort et seront éventuellement invités à saisir un mot de passe. Le nom du réseau et le mot de passe respectent la casse. Un réseau fermé peut renforcer la sécurité sur votre réseau.

Pour créer un réseau fermé :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur AirPort dans la barre d’outils, puis sur Sans fil.

  4. Cliquez sur « Options de réseau sans fil » et cochez la case « Créer un réseau fermé ».

Pour vous connecter à un réseau fermé, choisissez Autre dans le menu d’état AirPort situé dans la barre des menus, saisissez le nom du réseau et si nécessaire, le mot de passe correspondant. Si vous ne connaissez pas le nom ou le mot de passe du réseau, demandez à votre administrateur réseau.
