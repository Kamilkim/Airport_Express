Partage d’une imprimante sur le port WAN Ethernet

Si vous configurez votre borne d’accès pour partager une imprimante via le port WAN Ethernet, des ordinateurs situés en dehors de votre réseau local pourront l’utiliser pour imprimer. Par exemple, si vous êtes au bureau, vous pouvez imprimer sur l’imprimante USB connectée à la borne d’accès de votre domicile.

Pour partager une imprimante sur le port WAN Ethernet :

  1. Connectez votre imprimante USB à votre borne d’accès AirPort ou à votre disque Time Capsule.

  2. Ouvrez l’Utilitaire AirPort qui se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac ou dans Démarrer > Tous les programmes > AirPort sur un PC fonctionnant sous Windows.

  3. Sélectionnez le périphérique qui va partager l’imprimante, puis cliquez sur Configuration manuelle.

  4. Cliquez sur Imprimantes dans la barre d’outils, puis sélectionnez « Partager des imprimantes via le port WAN Ethernet ».
