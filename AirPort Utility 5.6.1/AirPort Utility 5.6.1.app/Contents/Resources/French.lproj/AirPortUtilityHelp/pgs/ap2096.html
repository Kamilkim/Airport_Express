Contrôle du champ de détection de votre réseau

Vous pouvez limiter le champ de détection de votre réseau AirPort, afin de contrôler les personnes qui peuvent y accéder, en réduisant le niveau du réglage de puissance de transmission. Lorsque vous limitez le champ de détection du réseau, seuls les ordinateurs situés suffisamment près de la borne d’accès ou du disque Time Capsule peuvent accéder au réseau.

Pour définir la puissance de transmission :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous souhaitez contrôler, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur AirPort dans la barre d’outils, puis sur Sans fil.

  4. Cliquez sur « Options de réseau sans fil », puis choisissez un réglage dans le menu local Puissance de transmission (plus le pourcentage est faible, plus le champ de détection du réseau est restreint).
