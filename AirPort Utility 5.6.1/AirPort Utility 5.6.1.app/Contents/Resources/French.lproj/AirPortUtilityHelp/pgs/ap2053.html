Connexion à Internet via Ethernet

Si, pour vous connecter à Internet, vous utilisez un modem DSL ou câble ou encore un réseau local (LAN), vous pouvez connecter votre borne d’accès ou votre disque Time Capsule au modem ou au réseau et partager la connexion Internet avec des ordinateurs et des clients de votre réseau sans fil.

Vous devez configurer le mode d’attribution d’une adresse Internet à votre borne d’accès ou disque Time Capsule de la part de votre fournisseur d’accès à Internet (FAI) ou de votre réseau, ainsi que la façon dont le périphérique partage la connexion à Internet avec les ordinateurs du réseau.

Ouvrez Utilitaire AirPort, sélectionnez le périphérique à configurer, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe. Cliquez sur Internet dans la barre d’outils, puis sur Connexion à Internet. Choisissez Ethernet dans le menu local Connexion via, puis cliquez sur TCP/IP.

Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

Pour configurer votre périphérique pour une connexion via Ethernet :

  1. Configurez le mode d’attribution de l’adresse IP par votre FAI pour le périphérique sans fil, à l’aide du menu local Configurer IPv4 en choisissant l’un des éléments suivants :

    • Choisissez Via DHCP si votre FAI vous attribue une adresse IP automatiquement.

    • Choisissez Manuellement si votre FAI vous a attribué une adresse IP en même temps que votre compte. Saisissez les informations que vous avez reçues de votre FAI, par exemple, votre adresse IP, le masque de sous-réseau, l’adresse du routeur et toute autre donnée nécessaire.

  2. Cliquez sur Partage de connexion et configurez la manière dont votre périphérique partage la connexion Internet avec les ordinateurs de votre réseau, ou configurez le périphérique en tant que pont s’il est connecté à un réseau.

    • Choisissez « Partager une adresse IP publique » dans le menu local Partage de connexion pour utiliser DHCP ou NAT dans le but de partager votre adresse IP avec les ordinateurs de votre réseau.

    • Choisissez « Distribuer une plage d’adresses IP » dans le menu local Partage de connexion si votre FAI vous a attribué une plage d’adresses IP.

    • Choisissez Désactivé (mode pont) si le périphérique est connecté à un réseau sur lequel un serveur DHCP distribue les adresses IP.

  3. Cliquez sur Mettre à jour pour réinitialiser le périphérique avec ces nouveaux réglages.

Si vous vous connectez à Internet à l’aide d’un compte PPPoE (Point-to-Point Protocol over Ethernet, ou protocole point-à-point via Ethernet), cliquez sur le lien ci-dessous.
