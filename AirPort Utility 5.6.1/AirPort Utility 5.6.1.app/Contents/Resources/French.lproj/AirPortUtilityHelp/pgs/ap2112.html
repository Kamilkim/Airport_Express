Utilisation de votre borne d’accès pour vous connecter à Internet

Si votre borne d’accès est fournie avec un modem interne et que vous disposez d’un compte pour une connexion commutée avec un fournisseur d’accès à Internet (FAI), vous pouvez brancher le port du modem de la borne d’accès sur une ligne téléphonique classique et utiliser le protocole PPP pour vous connecter à Internet.

Assurez-vous de disposer des informations sur votre compte avant d’utiliser Utilitaire AirPort.

Les informations suivantes peuvent s’avérer nécessaires :

  • le nom de votre compte auprès de votre FAI (à savoir votre nom d’utilisateur) et votre mot de passe ;

  • le numéro de téléphone de votre FAI pour votre modem commuté ;

  • si vous voulez configurer votre TCP/IP manuellement ou via PPP ;

  • votre adresse IP (configuration manuelle uniquement) ;

  • l’adresse de votre serveur DNS (Domain Name System).

Cliquez sur Modem pour régler les options du modem, comme la durée d’inactivité du modem avant la déconnexion, le code de votre pays ou autres options.
