Configuration du contrôle d’accès RADIUS

Le protocole RADIUS (Remote Authentication Dial-In User Service) est un protocole client-serveur qui permet à des serveurs d’accès à distance d’authentifier les utilisateurs, accédant sans fil au réseau, à l’aide d’une base de données centrale, ce qui permet d’éviter de devoir maintenir des informations relatives aux utilisateurs sur chacun des périphériques du réseau.

Une fois le serveur RADIUS configuré (reportez-vous à la documentation fournie avec le serveur), ouvrez Utilitaire AirPort, sélectionnez votre périphérique sans fil et choisissez Borne d’accès > Configuration manuelle. Le cas échéant, saisissez le mot de passe. Cliquez sur AirPort dans la barre d’outils, puis sur Contrôle d’accès.

Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

Pour utiliser l’authentification RADIUS :

  1. Choisissez RADIUS dans le menu local « Contrôle d’accès d’adresses MAC ».

  2. Choisissez Par défaut ou Autre dans le menu local « Type de RADIUS » selon la configuration du serveur RADIUS.

  3. Saisissez l’adresse IP et toute autre information relative au serveur RADIUS. Si vous n’êtes pas certain des informations à saisir, contactez l’administrateur serveur RADIUS.
