Contrôle de l’accès d’un utilisateur à votre réseau

Vous pouvez contrôler l’heure et les jours où un ordinateur peut accéder à votre réseau AirPort Extreme. Vous pouvez déterminer quand un utilisateur peut utiliser le réseau pour accéder à Internet, partager des fichiers ou jouer à des jeux.

Pour configurer un accès programmé à votre réseau :

  1. Ouvrez Utilitaire AirPort qui se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac ou dans Démarrer > Tous les programmes > AirPort sur un PC fonctionnant sous Windows.

  2. Sélectionnez votre borne d’accès ou votre disque Time Capsule, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur AirPort dans la barre d’outils, puis sur Contrôle d’accès.

  4. Cliquez sur le bouton Ajouter (+) et saisissez l’adresse MAC et la description ou le nom des ordinateurs que vous autorisez à accéder au réseau. Vous pouvez également cliquer sur Cet ordinateur pour ajouter l’adresse MAC et le nom de l’ordinateur que vous utilisez.

  5. Choisissez une valeur dans les menus locaux. Choisissez un jour de la semaine ou Tous les jours dans le menu local des jours, puis choisissez soit « toute la journée » soit « entre » dans le menu local annexe. Si vous avez choisi « entre », vous pouvez modifier les heures du jour en double-cliquant dans les champs des heures.

Si aucun ordinateur n’est répertorié, vous pouvez adapter le comportement par défaut d’un utilisateur accédant au réseau en cliquant sur Modifier et en choisissant les valeurs qui s’imposent dans les menus locaux.
