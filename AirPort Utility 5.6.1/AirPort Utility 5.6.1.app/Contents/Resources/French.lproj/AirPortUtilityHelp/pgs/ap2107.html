Configuration de comptes utilisateur pour des disques connectés à un réseau AirPort

Vous pouvez configurer des comptes destinés aux utilisateurs de votre réseau sans fil afin de contrôler comment un utilisateur partage des disques durs USB connectés à votre borne d’accès. Si vous utilisez un disque Time Capsule, vous pouvez configurer des comptes pour les utilisateurs du disque AirPort interne.

Pour configurer un compte utilisateur :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Disques dans la barre d’outils, puis sur Partage de fichiers.

  4. Choisissez « Avec les comptes » dans le menu local « Sécuriser les disques partagés », puis cliquez sur Configurer les comptes.

  5. Cliquez sur Ajouter (+), tapez le nom de l’utilisateur et attribuez un mot de passe à ce dernier.

  6. Dans le menu local « Accès au partage de fichier », choisissez parmi les valeurs Lecture et écriture, Lecture seule ou Non autorisé, en fonction des privilèges d’accès que vous voulez attribuer à cet utilisateur.
