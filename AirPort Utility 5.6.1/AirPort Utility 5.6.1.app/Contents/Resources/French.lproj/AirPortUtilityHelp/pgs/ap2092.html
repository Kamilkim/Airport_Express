Affichage des informations relatives à votre borne d’accès ou disque Time Capsule

Vous avez la possibilité de consulter les réglages de votre borne d’accès ou de votre disque Time Capsule, tels que l’adresse IP, le nom du réseau créé, la version du programme interne du périphérique, etc. Cette fonction peut vous aider à résoudre certains problèmes ou à partager des informations avec d’autres utilisateurs ou administrateurs du réseau.

Pour afficher le résumé des réglages de borne d’accès :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous souhaitez utiliser, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Double-cliquez sur le titre d’un réglage pour accéder à la sous-fenêtre correspondante dans Utilitaire AirPort.

Si vous utilisez Utilitaire AirPort sur un Mac, vous pouvez également taper un mot ou une expression dans le champ de recherche et choisir un résultat dans la liste pour accéder à cette sous-fenêtre de l’utilitaire.
