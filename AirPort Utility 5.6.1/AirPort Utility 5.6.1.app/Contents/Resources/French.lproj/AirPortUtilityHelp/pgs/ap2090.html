Recherche de réglages AirPort

Si vous utilisez Utilitaire AirPort sur un ordinateur Macintosh, vous pouvez recourir au champ de recherche proposé dans Utilitaire AirPort pour retrouver rapidement le réglage qui vous intéresse, tel que l’adresse IP ou le mode radio de votre borne d’accès ou de votre disque Time Capsule.

Pour rechercher un réglage particulier dans Utilitaire AirPort :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Tapez un mot ou une expression dans le champ de recherche.

  4. Sélectionnez le réglage dans la liste pour y accéder directement.
