Configuration de votre réseau AirPort

Utilisez la sous-fenêtre Sans fil pour régler les options de votre réseau sans fil.

Les options sans fil comprennent les éléments suivants :

  • Configurez le mode sans fil pour créer un nouveau réseau sans fil ou étendre un réseau sans fil.

  • Attribution d’un nom à votre réseau. Il s’agit du nom qui apparaît dans le menu d’état AirPort situé dans la barre des menus sur les ordinateurs Mac OS X et au-dessus de l’icône de réseau sans fil qui se trouve dans la barre d’état des ordinateurs sous Windows.

  • Réglage du mode radio. Choisissez pour votre réseau sans fil la fréquence la plus compatible avec les ordinateurs prévus pour accéder à votre réseau.

    • Si vous pensez que les ordinateurs utiliseront une connexion 802.11b ou 802.11g pour accéder à votre réseau, choisissez « 802.11n (compatible 802.11b/g) » dans le menu local Mode radio.

    • Pour que des ordinateurs utilisant une connexion 802.11a puissent se joindre à votre réseau, choisissez « 802.11n (compatible 802.11a) ».

    • Pour que seuls les ordinateurs utilisant une connexion 802.11n puissent se joindre à votre réseau, choisissez « 802.11n uniquement (2,4 GHz) » ou « 802.11n uniquement (5 GHz) » dans le menu local Mode radio.

  • Choix d’un canal pour votre réseau. Dans la plupart des cas, le réglage Automatique est le plus efficace, bien qu’il existe des cas où il est préférable de choisir un canal spécifique pour votre réseau. Si vous utilisez 802.11n uniquement dans une bande passante de 5 GHz, il est impossible de régler manuellement les canaux.

  • Protection du votre réseau à l’aide d’un mot de passe. Choisissez soit WEP (Wired Equivalent Privacy), WPA (Wi-Fi Protected Access) ou WPA2 Personnel, WPA ou WPA2 Entreprise, soit WEP (réseau de sécurité transitionnel), en fonction de la borne d’accès que vous configurez et des caractéristiques des ordinateurs clients sans fil prévus pour accéder à votre réseau.

  • Réglage d’options supplémentaires pour l’accès sans fil. Cliquez sur « Options de réseau sans fil » sans fil pour régler le taux de multidiffusion, la puissance de transmission ou la région, pour créer un réseau fermé et pour activer la robustesse d’interférence. Si votre borne d’accès le permet, et si vous utilisez le mode radio 802.11n uniquement (5 GHz), vous pouvez également sélectionner l’utilisation des canaux larges qui fournissent un débit de données plus important sur votre réseau.
