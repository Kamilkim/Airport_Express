Contrôle de la durée de validité de l’adresse IP d’un ordinateur

Pour contrôler la durée de validité d’une adresse IP pour un ordinateur de votre réseau AirPort, réglez la durée de bail DHCP.

Pour régler la durée du bail DHCP :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Internet dans la barre d’outils, puis choisissez « Partager une adresse IP publique » ou « Distribuer une plage d’adresses IP » dans le menu local Partage de connexion, puis cliquez sur DHCP.

  4. Saisissez un nombre dans le champ Bail DHCP, puis choisissez le nombre de minutes, d'heures ou de jours dans le menu local.

Si vous utilisez des durées de bail très courtes, le protocole DHCP peut reconfigurer de façon dynamique les réseaux dans lesquels le nombre d’ordinateurs est supérieur au nombre d’adresses IP disponibles.

Si vous avez configuré des réservations DHCP, les ordinateurs de la liste n’obtiennent alors pas de nouvelle adresse DHCP à l’expiration du bail DHCP.
