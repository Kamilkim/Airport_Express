Attribution d’un nom et d’un mot de passe à votre borne d’accès ou à votre disque Time Capsule

Attribuez un nom à votre périphérique sans fil pour pouvoir l’identifier facilement si plusieurs périphériques se trouvent sur votre réseau. Cela permet de localiser facilement le périphérique. Le nom du périphérique sans fil apparaît alors dans la liste de gauche d’Utilitaire AirPort. Vous pouvez également protéger votre périphérique sans fil à l’aide d’un mot de passe. Cela permet de s’assurer que seules les personnes qui connaissent le mot de passe peuvent modifier les réglages.

Les informations relatives aux contacts et à l’emplacement peuvent être utiles si vous utilisez le protocole SNMP (Simple Network Management Protocol) pour contrôler l’activité de votre périphérique.

Cochez la case « Autoriser la configuration via WAN » pour configurer vos périphériques sans fil via le port WAN Ethernet. Cela vous permet de modifier des réglages tout en étant connecté à un réseau. Si votre périphérique sans fil le permet et que vous disposez d’un compte et d’un service de noms de domaine, vous pouvez cocher la case « Permettre la configuration sur Internet via Bonjour » pour le rendre accessible via Internet.

Si vous avez accès à un serveur d’horloge de réseau, que ce soit de votre réseau ou d’Internet, vous pouvez cocher la case « Régler automatiquement la date et l’heure » et sélectionner un serveur d’horloge de réseau. Si cette fonction est compatible avec votre périphérique, vous pouvez également choisir un fuseau horaire dans le menu local. Cela garantit que votre périphérique est toujours à l’heure, ce qui est très utile si vous contrôlez des historiques.
