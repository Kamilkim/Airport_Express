Acheminement du trafic réseau vers un ordinateur donné de votre réseau

Pour garantir un acheminement correct à travers votre réseau des demandes vers votre serveur web, AppleShare ou FTP, ou vers un ordinateur donné de votre réseau, vous devez définir une adresse IP permanente pour votre serveur ou ordinateur et fournir à la borne AirPort ou au disque Time Capsule les informations de mappage des ports entrants.

Pour configurer le mappage des ports entrants :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Avancé dans la barre d’outils, puis sur Mappage de port.

  4. Cliquez sur le bouton Ajouter (+) et choisissez un service dans le menu local Service (par exemple, Partage de fichiers personnel).

  5. Saisissez l’adresse IP du serveur dans le champ « Adresse IP privée ».

Pour utiliser le mappage de ports, vous devez configurer manuellement le port TCP/IP sur l’ordinateur utilisé comme serveur Web, AppleShare ou FTP.
