Importation de réglages

Si vous avez enregistré dans un fichier les réglages d’une borne d’accès ou d’un disque Time Capsule ou que l’administrateur réseau vous en a fourni un, vous pouvez importer les réglages de la borne d’accès ou du disque Time Capsule.

Vous pouvez importer l’ensemble des réglages ou sélectionner le contrôle d’accès programmé, les réglages de compte ou tous les autres réglages.

Pour importer les réglages de la borne d’accès :

  1. Ouvrez Utilitaire AirPort qui se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac ou dans Démarrer > Tous les programmes > AirPort sur un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Choisissez Fichier > Importer, localisez le fichier de réglages, puis cliquez sur Ouvrir.

  4. Sélectionnez les réglages à importer, puis cliquez sur OK.

  5. Cliquez sur Mettre à jour pour activer les nouveaux réglages.

Pour enregistrer un fichier de réglages, choisissez Fichier > « Enregistrer une copie sous ». Sélectionnez « Inclure les mots de passe comme données chiffrées » pour obliger à saisir le mot de passe du périphérique au moment d’importer les réglages ultérieurement. Pour afficher tous les réglages hormis les mots de passe, vous pouvez cliquer sur Continuer sans avoir à taper de mot de passe.
