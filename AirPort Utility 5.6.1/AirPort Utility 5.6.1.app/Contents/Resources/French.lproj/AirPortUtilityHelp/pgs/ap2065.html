Journalisation des informations du réseau sans fil

Vous pouvez configurer votre borne d’accès AirPort ou votre disque Time Capsule de manière à consigner les informations d’état dans l’historique système de Mac OS X (ou hôte syslog) sur un ordinateur utilisant Mac OS X, ou via une application syslog sur un ordinateur utilisant Windows. Cela peut-être utile pour résoudre les problèmes et surveiller l’activité du périphérique sans fil.

Pour garantir la précision des historiques, la date et l’heure de base du périphérique sans fil doivent être définies automatiquement.

Pour configurer la journalisation de la borne d’accès :

  1. Ouvrez Utilitaire AirPort qui se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac ou dans Démarrer > Tous les programmes > AirPort sur un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Le cas échéant, saisissez le mot de passe.

  3. Cliquez sur Avancé dans la barre d’outils, puis sur « Journalisation et Statistiques ».

  4. Saisissez l’adresse IP de l’ordinateur destiné à héberger les historiques de la borne d’accès dans le champ « Adresse de destination Syslog ».

  5. Choisissez le type d’informations à enregistrer dans le menu local Niveau Syslog.

  6. Pour consulter les historiques de la borne d’accès, cliquez sur Historiques et statistiques.

Pour que les historiques contiennent des informations justes concernant la date et l’heure, sélectionnez Régler l’heure automatiquement dans la sous-fenêtre Borne d’accès des réglages AirPort, puis choisissez un serveur d’horloge de réseau et un fuseau horaire dans les menus locaux.
