Partage de votre connexion à Internet

Vous pouvez configurer votre borne d’accès ou votre disque Time Capsule de manière à partager une même adresse IP entre tous les clients du réseau via le protocole DHCP (Dynamic Host Configuration Protocol) et la traduction d’adresses réseau (NAT).

Si votre borne d’accès ou votre disque Time Capsule se connecte à Internet via PPPoE, il ou elle partage déjà une adresse IP via DHCP et NAT.

Pour partager votre connexion à Internet :

  1. Ouvrez Utilitaire AirPort situé dans le dossier Applications/Utilitaires dans le cas d’un ordinateur fonctionnant sous Mac OS X, ou dans Démarrer > Tous les programmes > AirPort pour un PC fonctionnant sous Windows.

  2. Sélectionnez le périphérique que vous configurez, puis cliquez sur Configuration manuelle. Si nécessaire, tapez le mot de passe de la borne d’accès.

  3. Cliquez sur Internet dans la barre d’outils, puis sur Connexion à Internet.

  4. Choisissez « Partager une adresse IP publique » ou « Distribuer une plage d’adresses IP » dans le menu local Partage de connexion.

L’activation du protocole DHCP et NAT via le partage d’une adresse IP publique, ou la distribution d’une plage d’adresses IP permet à votre borne d’accès ou à votre disque Time Capsule d’affecter dynamiquement et automatiquement des adresses IP aux ordinateurs clients, ce qui simplifie la configuration TCP/IP de chaque ordinateur.
