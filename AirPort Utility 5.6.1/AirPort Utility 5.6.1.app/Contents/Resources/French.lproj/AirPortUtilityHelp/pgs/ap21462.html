Archivage du contenu de Time Capsule

Chaque Time Capsule est dotée d’un disque AirPort interne dont vous pouvez archiver le contenu à l’aide d’Utilitaire AirPort.

Pour procéder à l’archivage du contenu de Time Capsule, vous devez brancher un disque externe sur le port USB de votre Time Capsule.

Pour archiver le contenu de Time Capsule :

  1. Ouvrez l’Utilitaire AirPort qui se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac ou dans Démarrer > Tous les programmes > AirPort sur un PC fonctionnant sous Windows.

  2. Sélectionnez la Time Capsule qui contient le disque AirPort interne que vous souhaitez archiver, puis cliquez sur Configuration manuelle.

  3. Cliquez sur Disques dans la barre d’outils, sélectionnez le disque interne AirPort, puis choisissez Archiver un disque dans le menu local.

  4. Choisissez le disque de destination dans le menu local Destination, puis cliquez sur Archiver.

Le voyant d’état de Time Capsule orange clignote pendant l’archivage du disque. Vous ne pourrez pas modifier les réglages, mettre à jour le programme interne ou redémarrer la Time Capsule tant que le contenu du disque n’est pas archivé et que la Time Capsule n’est pas à nouveau disponible.
