Effacement d’un disque Time Capsule

Si vous utilisez un disque Time Capsule, vous pouvez utiliser Utilitaire AirPort pour effacer le contenu de son disque.

Au moment d’effacer le disque AirPort interne, vous pouvez choisir les niveaux de sécurité suivants :

Effacement rapide (non sécurisé) : cette option permet d’effacer les informations des répertoires du disque afin que les données ne soient plus accessibles. Les données restent sur le disque jusqu’à ce que de nouvelles données viennent les écraser. Les données étant potentiellement récupérables, cette option est la moins sûre.

Mise à zéro des données : choisissez cette option si vous souhaitez remplacer toutes les données du disque par des zéros. Cette option offre une bonne protection des données en un minimum de temps. Comme cette option écrit sur tout le disque, elle prend plus de temps que la précédente.

Effacement (7 passes) : cette option remplace sept fois de suite les données du disque par des zéros. Elle offre un effacement des données hautement sécurisé, mais prend sept fois plus de temps que l’option « Mise à zéro des données ».

Effacement (35 passes) : choisissez cette option pour effacer le disque de la manière la plus sécurisée. Cette option remplace 35 fois de suite les données du disque par des zéros. C’est la méthode la plus sûre, mais elle prend 35 fois plus de temps que l’option « Mise à zéro des données ».

Pour effacer le disque AirPort interne de Time Capsule :

  1. Ouvrez l’Utilitaire AirPort qui se trouve dans le dossier Applications/Utilitaires dans le cas d’un Mac ou dans Démarrer > Tous les programmes > AirPort sur un PC fonctionnant sous Windows.

  2. Sélectionnez le disque Time Capsule qui contient le disque AirPort interne que vous souhaitez effacer, puis cliquez sur Configuration manuelle.

  3. Cliquez sur Disques dans la barre d’outils, sélectionnez le disque interne AirPort, puis choisissez Effacer un disque dans le menu local.

  4. Saisissez un nom pour le disque, choisissez un niveau de sécurité dans le menu local Effacement sécurisé, puis cliquez sur Effacer.

Le voyant d’état de Time Capsule clignote en orange pendant l’effacement du disque. Vous ne pourrez pas modifier les réglages, mettre à jour le programme interne ou redémarrer Time Capsule avant que le disque ne soit effacé et ne soit de nouveau disponible.
